keyword.URL value resets itself upon restart of Firefox to FreeCause.

Build Identifier: Mozilla/5.0 (Macintosh; U; Intel Mac OS X 10.6; en-US; rv:1.9.2.4) Gecko/20100413 Firefox/3.6.4

when changing the default search engine using the Location bar in "about:config" under "keyword.URL", the value resets itself to FreeCause search every time Firefox is restarted.

Reproducible: Always

Steps to Reproduce:
1.go to about:config
2.use the filter to filter to "keyword.URL"
3.change the string to "http://www.google.com/search?q="
4.restart firefox

Actual Results:  
string was reset to freecause, location bar search will lead to FreeCause search instead of Google.

Expected Results:  
Should have stayed on google, using the I'm Feeling Lucky function.
