setting home page with "Use Current Page" uses first tab, not active tab
User-Agent:       GoogleBot/2.1 (+http://www.googlebot.com/bot.html)
Build Identifier: Firefox 1.0

setting home page with "Use Current Page" button uses the first tab, not the
active tab

Reproducible: Always
Steps to Reproduce:
1. Open a few tabs with different pages.  
2. Select the right-most tab
3. Tools -> Options -> General -- Click on "Use Current Page"
Actual Results:  
The URL from the left-most tab was used.

Expected Results:  
It should have used the URL from the active tab.
