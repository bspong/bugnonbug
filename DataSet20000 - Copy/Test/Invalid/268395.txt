FF lacks preference settings for where internal and external links should be opened
User-Agent:       Mozilla/5.0 (Macintosh; U; PPC Mac OS X Mach-O; rv:1.7.3) Gecko/20040913 Firefox/0.10.1
Build Identifier: Mozilla/5.0 (Macintosh; U; PPC Mac OS X Mach-O; rv:1.7.3) Gecko/20040913 Firefox/0.10.1

I request new preference settings to specify behavior when clicking on links
internal to the web site on which they appear (lacking the "http://" in the
anchor tag"), and also to specify on behavior when clicking on links external to
the web site on which they appear (including the "http://" in the anchor tag").

Example:

When clicking on a link to a page within the current web site:
o Replace current page
o Open new page in new background tab
o Open new page in new foreground tab
o Open new page in new window

When clicking on a link to a page external to the current web site:
o Replace current page
o Open new page in new background tab
o Open new page in new foreground tab
o Open new page in new window

When clicking on a link to search results in one of my search engines:
o Replace current page
o Open new page in new background tab
o Open new page in new foreground tab
o Open new page in new window
