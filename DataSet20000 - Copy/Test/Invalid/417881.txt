Go button does not appear when Location Bar gets focus
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9b3) Gecko/2008020514 Firefox/3.0b3
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9b3) Gecko/2008020514 Firefox/3.0b3

When the URL in the Location Bar is selected(the entire or part of the URL), the Go button is not displayed.

The Reload button is useful in most situations. However, in some situations the user might wish to navigate to the URL displayed in the Location Bar, instead of reloading the current page. (I.e. the URL in the Location Bar is not always the same as the current page.)

Reproducible: Always

Steps to Reproduce:
1.Enter a URL in the Location Bar.
2.Press Enter or click the Go button.
3.Click or drag to select the URL in the Location Bar.
Actual Results:  
The Go button is not displayed in the Location Bar.

Expected Results:  
The Go button is displayed in the Location Bar.
