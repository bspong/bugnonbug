Unstable random moving of a tab while open to a new window very frequently.

On multiple computers using Windows XP & Vista Firefox before and after update Firefox 6 browser randomly closes a tab while the tab is open to move content to a new window.  This frequently occurs.  It delays whatever is being done.

Reproducible: Sometimes

Steps to Reproduce:
1.Open existing tab for use.  
2.Any time from opening until leaving tab it is subject to closing and having content moved to a new window.
3.
Actual Results:  
Randomly, Any time from opening a tab until leaving that tab it is subject to closing and having content moved to a new window.

Expected Results:  
Tab opens content used and tab is left with tab remaining in place.

Instability has required planned work using other browsers other than Firefox. This was initially reported on single computer using Widows XP - It has now been found to occur on multiple computers with both Windows XP & Windows Vista.  all browser windows may disappear when the new window is minimized.
