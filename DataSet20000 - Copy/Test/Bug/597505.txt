While browsing, memory usage keeps going up until reaching maximum amount for the system.

Symptoms: Firefox gets gradually slower along with all programs, running out of memory.

For me, it happens while 10-11 tabs are open.

Reproducible: Always

Steps to Reproduce:
1. Open Firefox
2. Load some tabs
3. Browse as usual, until Firefox gets terribly slow
Actual Results:  
Memory usage raise up to around 500.000 KB (my RAM is 768 MB)
All programs runs very slow or not responding.

Expected Results:  
Average use of memory at around 30.000-50.000 KB.
No lags and slowdowns.
