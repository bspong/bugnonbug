Visual glitching with page elements in 1.5.0.3 during scroll
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8.0.3) Gecko/20060426 Firefox/1.5.0.3
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8.0.3) Gecko/20060426 Firefox/1.5.0.3

Upon viewing any page, random elements will be sheared vertically by 1-2 pixels.  These elements can me images, form fields, etc.  The problem fixes itself when the user scrolls the page.  It will not be fixed visually after EACH scroll, but sooner or later, one of the scrolls will remove the visual shearing/distoring from the image.

Reproducible: Always

Steps to Reproduce:
1. Load up a webpage that allows scrolling vertically
2. Randomly, you'll find an element that is sheared by a couple pixels
3. Scroll up/down until randomly the shearing goes away.

Actual Results:  
Visual shearing came and went.

Expected Results:  
No visual shearing should result from simple page scrolling.

I have tried this bug in safe mode, and it is still present.  When I view the exact same pages in MSIE v6, I can scroll up/down as often as I want and there never is any actual element shearing.
