Verisign Certificate is not recognized by FireFox 1.0
Our web site has a new Verisign Certificate that is not recognizable by Firefox
1.0. The fix was to import the below Verisign certicifacte to the authorites tab
of the Manage Certificate window.

Uninstalling and install Firefox does not fix the problem. I also tried setting
different settings under the Security and Validation dropdowns in the Advanced
Options window. Only adding the certificate seems to fix the problem.

I have verified this on several Windows 2000 and Sun systems. You can see the
problem by going to https://www.getty.edu/mygetty/
