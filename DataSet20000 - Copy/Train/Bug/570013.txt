Mac OSX Minefield 3.7a5pre 2010-06-03 fails to open properly
User-Agent:       Mozilla/5.0 (Macintosh; U; Intel Mac OS X 10.6; en-US; rv:1.9.3a5pre) Gecko/20100505 Minefield/3.7a5pre
Build Identifier: Mozilla/5.0 (Macintosh; U; Intel Mac OS X 10.6; en-US; rv:1.9.3a5pre) Gecko/20100505 Minefield/3.7a5pre

On Mac OSX 10.6.3 (snowleopard) on Core 2 Duo Intel, all Minefield 3.7a5pre versions produce funny error window, partially obscured, on upper left of display.  Able to quit only.  This may have something to do with extensions, but, still happens with all extensions disabled.

Reproducible: Always

Steps to Reproduce:
Steps to Reproduce
---------------------------------------
1. Open any Minefield 3.7a5pre version from 5/19-6/3, 32-bit or 64-bit, with or without extensions
2. Get unreadable error message in upper left of screen
3.  Exit and go back to 5/5 version, which works!
Actual Results:  
What actually happened:
----------------------------------------
Instead, got a funny error message, partially obscured by the left edge of the display.  Error message is unreadable, but, may have something to do with extensions.  You get this even with no extensions enabled.  There are error messages in the system log.

Expected Results:  
What should have happened:
---------------------------------------
Should have gotten a web page window.

Environment: Intel Mac, Core 2 Duo, OSX 10.6.3 (fully patched Snowleopard).
