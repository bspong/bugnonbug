When I close my MacBook (OSX v10.5.7) it goes into standby as usual.  When I reopen it and put in my password, all programs work fine except Firefox (3.5.2).  It will minimize to the dock, the menus will open, but I can't interact with the page displayed and I can't switch tabs.  I can switch windows though (can't interact on any of them). I end up quitting Firefox and reopening it (tabs are saved correctly and reopened).  So the work around is obvious, but inconvenient.  And can't be worked around in Private Browsing.

Reproducible: Sometimes

Steps to Reproduce:
1. Open Firefox
2. Put computer in standby mode
3. Return from standby mode
Actual Results:  
Can't interact with the web pages and tabs.

Expected Results:  
Should be able to interact with the web pages and tabs.
