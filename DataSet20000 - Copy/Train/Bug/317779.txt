'Clear Private Data' causes AI RoboForm toolbar to go blank.
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8) Gecko/20051111 Firefox/1.5
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8) Gecko/20051111 Firefox/1.5

With one or any number of tabs open, and the RoboForm toolbar installed and visible, select:
  - Tools
  - Clear Private Data...
  - press 'Clear Private Data now'
The RoboForm toolbar will be present, but is now empty (no text or icons).
Turning the toolbar off and back on does not fix the problem... you must 
restart Firefox to get the RoboForm toolbar contents back to normal.

Reproducible: Always

Steps to Reproduce:
With one or any number of tabs open, and the RoboForm toolbar installed and visible, select:
  - Tools
  - Clear Private Data...
  - press 'Clear Private Data now'

Actual Results:  
Private data is cleared, and the RoboForm toolbar is present, but it is now empty (no text or icons). Turning the toolbar off and back on does not fix the problem... you must restart Firefox to get the RoboForm toolbar contents back to normal.

Expected Results:  
Private data is cleared, with no impact on the RoboForm toolbar.
