Right click on image, 'Set as Desktop Background' results in (null) Background
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 5.0; en-US; rv:1.8b3) Gecko/20050712 Firefox/1.0+
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.0; en-US; rv:1.8b3) Gecko/20050712 Firefox/1.0+

When setting a desktop background, the resulting image file is called '(null)
Background.bmp'

Reproducible: Always

Steps to Reproduce:
1. Open up any web page with an image.  
2. Right click on the image, and select 'Set as Desktop Background'.  
3. Proceed through the dialog.  
4. Go to your Desktop, and see that the new image is there.  
5. Right click on the desktop, choose 'Properties'
6. In the background tab, the current desktop background will be '(null) Background'
Actual Results:  
In the background tab, the current desktop background will be '(null) Background'

Expected Results:  
I would have expected the image to be called 'Deer Park Alpha 2 Background'

This is from Deer Park Alpha 2, so it might just be a branding thing.
