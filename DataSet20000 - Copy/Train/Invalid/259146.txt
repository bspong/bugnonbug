Gmail Invites blocked by popup blocker, even if popups allowed from gmail.google.com
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 5.1; rv:1.7.3) Gecko/20040911 Firefox/0.10 (bangbang023)
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; rv:1.7.3) Gecko/20040911 Firefox/0.10

Similar to bug 258499 but this occurs when you set Firefox to allow all popups
from gmail.google.com, it still blocks the invite window.

Reproducible: Always
Steps to Reproduce:
1. Add "gmail.google.com" to the Allow list in the popup blocker
2. Click "Invite a Friend" in Gmail

Actual Results:  
Popup blocker reports it blocked a popup.

Expected Results:  
Opened the invite window
