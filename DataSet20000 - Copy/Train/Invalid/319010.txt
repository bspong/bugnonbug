'browser.preferences.instantApply' option turns off 'OK' button in Tools -> Options
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8) Gecko/20051111 Firefox/1.5
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8) Gecko/20051111 Firefox/1.5

If you go into about:config, and turn the option 'browser.preferences.instantApply' to true, then go to Tools -> Options, the 'OK' button at the bottom of the window is no longer there.

Reproducible: Always

Steps to Reproduce:
1.Go to Tools -> Options, and see the three buttons down the bottom, 'OK', 'Cancel' and 'Help'
2.Open 'About:config' and change 'browser.preferences.instantApply' to true
3.Go to Tools -> Options again and see that there are now only two buttons there: 'Cancel' and 'Help'

Actual Results:  
'Ok' button is not there

Expected Results:  
'Ok' button is there (I guess)
