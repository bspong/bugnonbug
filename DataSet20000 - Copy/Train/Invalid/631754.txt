Downloads window closes when Firefox is closed - previous versions did not close window
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 6.1; en-US; rv:1.9.2.13) Gecko/20101203 Firefox/3.6.13
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 6.1; en-US; rv:1.9.2.13) Gecko/20101203 Firefox/3.6.13

The downloads window closed when Firefox is closed. Previous versions of Firefox kept the window open when Firefox closed. Useful when you are downloading large files, and you want to close Firefox, but keep the downloads window open.

Reproducible: Always

Steps to Reproduce:
1. Open Firefox
2. Open the Downloads window (Ctrl+J)
3. Close Firefox
Actual Results:  
Firefox and the Downloads window closed

Expected Results:  
Firefox closes, but leaves downloads window open
