Anonymizer Total Net Security: for Firefox, Netscape 7.2, IE 6, RSS

1. Install 1.0.4 with Anonymizer Total Net Security already installed,
configured (connecting to internet) as well as Firefox 1.0.3 already installed.

Actual Results:  
Problem vanished.

Expected Results:  
1.0.4 should have connected as usual, but with the added security fixes.
