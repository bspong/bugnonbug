QOOP uploader crashes Firefox
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 6.0; en-US; rv:1.9.1.3) Gecko/20090824 Firefox/3.5.3 (.NET CLR 3.5.30729)
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 6.0; en-US; rv:1.9.1.3) Gecko/20090824 Firefox/3.5.3 (.NET CLR 3.5.30729)

Clicking 'browse' in the QOOP uploader (which appears to be written in Javascript) crashes Firefox immediately.

Reproducible: Always

Steps to Reproduce:
1. You'll need to a create an account at qoop.com
2. Click 'upload' underneath 'Sell your images'
3. Click the 'browse' button on the page that appears
4. Firefox should crash
Actual Results:  
Firefox crashes

Expected Results:  
A file browse dialog should appear.
