No ability to uncheck "Ask me before clearing private data" in Privacy/Settings menu
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8.0.4) Gecko/20060508 Firefox/1.5.0.4
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8.0.4) Gecko/20060508 Firefox/1.5.0.4

When exiting Firefox, a confirmation dialog pops up about clearing private data. In the Privacy Settings menu, the option to turn off this overly annoying confirmation box is not accessible (shaded out). This bug has been around for the past 3 or 4 versions (updates).

Thanks,
Dan

Reproducible: Always

Steps to Reproduce:
1. Run Firefox / Tools / Options / Privacy tab / Settings / check Clear Private Data When Closing Firefox (cannot uncheck "Ask" box below that)
2. Close Firefox
3. Voila, confirmation box




Please enable the option to uncheck the "Ask me before clearing private data" in Privacy/Settings menu.
