User-Agent:       Mozilla/5.0 (Windows NT 5.1; rv:2.0.1) Gecko/20100101 Firefox/4.0.1
Build Identifier: Mozilla/5.0 (Windows NT 5.1; rv:2.0.1) Gecko/20100101 Firefox/4.0.1

Google 'Search' button doesn't work when text in the search field contains 'click' word.

Reproducible: Always

Steps to Reproduce:
1. Goto http://www.google.com and type 'click' in the search field or anything that contains 'click'.
2. Press 'Google Search' button or press Return key.


Actual Results:  
Nothing happens, search is not started.

Expected Results:  
Search results should be displayed.
