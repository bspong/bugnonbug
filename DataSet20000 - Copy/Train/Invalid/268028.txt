Titles will not disapear when the tab is changed while viewing them.
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 5.1; rv:1.7.3) Gecko/20041001 Firefox/0.10 StumbleUpon/1.998
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; rv:1.7.3) Gecko/20041001 Firefox/0.10 StumbleUpon/1.998

When the tab is changed while your pointer if over an element on a web page that
has a title attribute (usually images or hyperlinks), the text will not
disapear.  Some examples can be found at http://lolworld.ath.cx:99/ex1, 2, or 3.gif.

Reproducible: Sometimes
Steps to Reproduce:
1. Put your pointer over an element on a web page with a title atribute.
2. Change the tab while the text is still visible.
3. The text will not disapear.
Actual Results:  
The text will not disapear, and the browser window must be closed and reopened
to make it go away.

Expected Results:  
The text should have disapeared when the tab was changed or when the mouse was
moved.
