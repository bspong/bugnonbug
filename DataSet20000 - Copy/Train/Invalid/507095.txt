It's easy hanging the desktop if there are orca screen reader and firefox3.5.1 running together.
Steps to reproduce:
1. Start orca from gnome-terminal with speech and brlmonitor enabled.
2. Start firefox3.5.1.
2. Press <Alt><Tab> to switch focus from orca main window to firefox3.5.1.

Bug observations:
Desktop hangs, it will not respond any keyboard/mouse event any more. This bug is visible on opensolaris desktop vermillion_120/solaris nevada 119 build.
