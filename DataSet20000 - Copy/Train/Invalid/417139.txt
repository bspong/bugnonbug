Location Bar is uneditable in popup windows
User-Agent:       Mozilla/5.0 (Macintosh; U; Intel Mac OS X 10.4; es-ES; rv:1.9b4pre) Gecko/2008021204 Minefield/3.0b4pre
Build Identifier: Mozilla/5.0 (Macintosh; U; Intel Mac OS X 10.4; es-ES; rv:1.9b4pre) Gecko/2008021204 Minefield/3.0b4pre

The location bar cannot be edited in sites that launch popup windows.   You can launch new tabs, bookmarks, other functionality, but editing the Location bar does not seem to work.

Reproducible: Always

Steps to Reproduce:
1. Install nightly build
2. Find a site with a popup window  (eg. http://www.marketwatch.com or http://msnbc.com and launch something with video)
3. Verify in popup window, location bar is uneditable.  However, can open a new tab using keystroke, but cant enter in an address.
Actual Results:  
Location bar uneditable in a popup window

Expected Results:  
Location bar editable in a popup window
