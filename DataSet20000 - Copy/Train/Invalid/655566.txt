Copy paste with key shortcuts works only once in Google Docs Spreadsheet
User-Agent:       Mozilla/5.0 (Windows NT 6.1; WOW64; rv:2.0.1) Gecko/20100101 Firefox/4.0.1
Build Identifier: Mozilla/5.0 (Windows NT 6.1; WOW64; rv:2.0.1) Gecko/20100101 Firefox/4.0.1

Copy/pasting cells with key shortcuts work only once in Google Docs Spreadsheet. The first time it works fine, when trying again it copies again the first cell.

When using google docs menu instead of the keyboard: Edit - copy and Edit - paste, it works fine.

Reproducible: Always

Steps to Reproduce:
1.go to docs.google.com, create a new spreadsheet
2.write data two different cells
3.copy one cell, paste it into an empty cell
4.copy the second cell, paste it into another empty cell

Actual Results:  
the content of the first cell that was copied is pasted on the empty cell

Expected Results:  
the content of the second cell that was copied should be pasted on the empty cell
