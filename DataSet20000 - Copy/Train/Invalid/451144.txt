Browser options for tabs not working.
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9.0.1) Gecko/2008070208 Firefox/3.0.1
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9.0.1) Gecko/2008070208 Firefox/3.0.1

Have the tab options: "When I open a link in a new tab switch to it" set to null, but Firefox switches to the tab despite setting.

Reproducible: Always

Steps to Reproduce:
1. Set option "When I open a link in a new tab switch to it" to null.
2. Have a tab open with link or URL.
3. Click URL
Actual Results:  
Focus switches to the newly opened tab.

Expected Results:  
Focus NOT switching to the newly opened tab.
