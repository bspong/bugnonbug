Thumbnail Expander 1.0 crashing minefield while moving  mouse away to have the full image disappear
User-Agent:       Mozilla/5.0 (Windows NT 5.1; rv:2.0b5pre) Gecko/20100828 Firefox/4.0b5pre
Build Identifier: Mozilla/5.0 (Windows NT 5.1; rv:2.0b5pre) Gecko/20100828 Firefox/4.0b5pre

Thumbnail Expander 1.0 is extension to show liked to thumbnail image in pop-up. Despite the fact that it officially not compatible with trunk it worked without problems all the time before current (Gecko/20100828 Firefox/4.0b5pre) build. Now it cause minefield to crash.

Reproducible: Always

Steps to Reproduce:
1. Install Thumbnail Expander 1.0, restart
2. Use it to expand any image thumbnails into full image
3. Move cursor away from expanded image to make expanded image to disappear

Actual Results:  
Minefield immediately crash

Expected Results:  
Expanded image should just disappear
