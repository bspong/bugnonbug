Reset firefox to default in Safemode does not reset most recently visited pages
User-Agent:       Mozilla/5.0 (Macintosh; U; Intel Mac OS X 10.5; en-US; rv:1.9.0.17) Gecko/2009122115 Firefox/3.0.17
Build Identifier: Mozilla/5.0 (Macintosh; U; Intel Mac OS X 10.5; en-US; rv:1.9.2) Gecko/20100105 Firefox/3.6

Visit some site and bookmark them. Then Reset Firefox by checking all the check boxes in Safe mode. It resets to Default settings. Firefox is reset to default, but the most Recently visited pages are not reset, they can be seen when firefox is launched again.

Reproducible: Always

Steps to Reproduce:
1. Quit Firefox from Menu.
2. Press Hold Option key and start firefox
3. Safe mode Window is launched
4. Check all the check boxes
5. Click on make changes and restart
Actual Results:  
After reset Firefox through safe and restarting the firefox. Firefox is restored to default, But the most recently visited pages are not reset and can be seen in Bookmarks Toolbar > Most Visited.


Expected Results:  
All the changes have to be reset and firefox should be restored to default.
