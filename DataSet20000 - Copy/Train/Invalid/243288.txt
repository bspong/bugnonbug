Going to yahoo finance and filling out something in the search the web box (e.g.
microsoft) will result in a new window opening although popups are blocked in
the preferences.

Reproducible: Always
Steps to Reproduce:
1. go to http://finance.yahoo.com/
2. fill up anything in the box called "search the web" (on the upper right corner)
3. click search
4) watch new window open even if tools-->options-->web features---->block popup
window is checked

Actual Results:  
Popup window opens

Expected Results:  
Popup window should not open

There are no exceptions set up in my block popups.
