Application menu disappears when Element Properties opened
User-Agent:       Mozilla/5.0 (Macintosh; U; Intel Mac OS X 10.5; en-US; rv:1.9b4pre) Gecko/2008021104 Minefield/3.0b4pre
Build Identifier: Mozilla/5.0 (Macintosh; U; Intel Mac OS X 10.5; en-US; rv:1.9b4pre) Gecko/2008021104 Minefield/3.0b4pre

The OS X Application Menu disappears (with the exception of the Application name) when the Element Properties window is opened.

Reproducible: Always

Steps to Reproduce:
1. Navigate to a webpage that contains at least one image.
2. Right click, or control-click on an <img> element to display the context-menu.
3. Select 'Properties'.
Actual Results:  
The Element Properties window should be displayed.

Expected Results:  
The Element Properties window is displayed.
The Application menu disappears, until either the Element Properties window is closed, or loses focus.
