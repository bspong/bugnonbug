Bookmarks not sorted in Bookmarks menu
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.7) Gecko/20040707 Firefox/0.9.2
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.7) Gecko/20040707 Firefox/0.9.2

After managing bookmarks and choosing View -> Sorted by Name, the bookmarks in
the bookmark manager are sorted, but the bookmarks in the bookmarks menu in the
main window are not sorted.

Reproducible: Always
Steps to Reproduce:
1.Create several bookmarks in unsorted name order.
2.Choose Bookmarks -> Manage Bookmarks in main window.
3.Choose View -> Sorted by Name in Bookmarks Manager window.
4.Click Bookmarks in main window menu and notice that the bookmarks are not sorted.

Actual Results:  
The bookmarks are not sorted.

Expected Results:  
The bookmarks should be sorted.
