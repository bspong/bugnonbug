The Gamefox Extension (various add-ons for the GameFAQs forums) is not compatible with Fire Fox 1.5.
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8) Gecko/20051111 Firefox/1.5
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8) Gecko/20051111 Firefox/1.5

After downloading the beta version of Firefox 1.5 the GameFox extension is seen as incompatible.

Reproducible: Always

Steps to Reproduce:
1. Try to download and install the GameFox extension for Firefox 1.5.
Actual Results:  
It was disabled as it wasn't compatible.

Expected Results:  
It should have installed and worked.
