F5 brings up "Open File" dialog instead of reloading page

Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8.1.5) Gecko/20070713 Firefox/2.0.0.5

Since upgrading to 2.0.0.5, pressing the F5 key on any page will display the "Open File" dialog instead of performing a reload.

Reproducible: Always

Steps to Reproduce:
1.Load any page
2.Press F5
Actual Results:  
Standard "Open File" dialog is displayed.

Expected Results:  
The page should reload.

I've duplicated this behavior in safe mode.
