When you close Firefox software, firefox process still running.
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US) AppleWebKit/534.16 (KHTML, like Gecko) Chrome/10.0.629.0 Safari/534.16
Build Identifier: Firefox 4b10

Error exiting Firefox, operating system OpenIndiana build 148 ( I do not 
know if the same error in other OS).
When you close Firefox software (if you close the window firefox) firefox process 
still running and the computer is very busy. Ons at that time can not be 
started again. It is necessary to wait long minutes for it to be completed 
the process of firefox, or use the command pkill firefox. After this may 
already be running again ons.

Reproducible: Always

Steps to Reproduce:
1. run firefox 4b10
2. close firefox 4b10 (close window)
3.
Actual Results:  
- window firefox is closed
- firefox process still running

Expected Results:  
- window firefox is closed
- firefox process is not running
