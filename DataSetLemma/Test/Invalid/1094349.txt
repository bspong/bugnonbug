Various rendering problem since introduction of D2D 1.1
Created attachment 8517602
Screenshot 2014-11-05 10.38.50.png

In response to the frequent crash from 1091903, I tried disabling D2D 1.1.

Disabling it via the use1_1 pref fix the crash but then the browser's rendering periodically breaks, with all my window becoming blank white. That state never recovers so I have to terminate firefox.

Disabling it by removing d2d 1.1 from the backends list(s) cause various bitmap in the UI to stop rendering; this is consistent across browser restarts and machine reboots. This reproduces on a clean profile.
