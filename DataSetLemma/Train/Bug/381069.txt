Firefox hang/crash when loading large innerHTML table with select element
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.2; en-US; rv:1.8.1.3) Gecko/20070309 Firefox/2.0.0.3
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.2; en-US; rv:1.8.1.3) Gecko/20070309 Firefox/2.0.0.3

Loading the html page in additional information with 1000 select element crashes/hangs firefox. Some computer report firefox a crashing with an Application error (one Macintosh) while other Macintoshes just see Firefox hang. A number of Windows computer that see this error (Windows XP) see the underlying OS, or at least explorer crash, which indicates this may be a security issue.

After I reproduce this issue, I must restart my computer to continue working a random other application lock up. I have no problem before I reproduce the issue.


I have provided a sample page which crash Firefox. Rendering 100 element on the sample page doe not produce the error--only rendering higher numbers. For instance, using an input of 1000 in the test page reproducibly crash on my machine, and 300 on a Macintosh in the office.


I have reproduced this issue on:
Firefox 1.5
Firefox 2.0
Gran Paradiso Alpha 1
Firefox Minefield From: http://ftp.mozilla.org/pub/mozilla.org/firefox/nightly/latest-trunk/ a of 05/17/2007.

I have reproduced on the following OS:
Windows XP Professional x64
Windows XP 32-bit
Macintosh OSX

For reference, the page render properly on Safari and IE7, but I have had other user report IE7 crash on this page a well.



Reproducible: Always

Steps to Reproduce:
1. Load the html specified in additional information
2. Type in 1000 in the input box and hit execute
3. Firefox will crash/hang. Sometimes this take out the underlying operating system.
Actual Results:
Firefox will either crash with an application error or hang and not reproduce the issue. The hang consumes 100% of one CPU (50% on my box).

Expected Results:
Rendered the page properly with the number of element defined in the input box.

<html>
<head>
<style type="text/css">
table {
border: 1px solid black;
margin: 3px;
}
</style>
<script type="text/javascript">
function run() {
var num = parseInt(document.getElementById("iterations").value);
var out = [];
for (var i = num; i >= 0; i--) {
out.push("<table><tr><td><select><option>" + i + "</option></select></td><td>" + i + "</td><td><input type='text' value='" + i + "'></td></tr></table>");
}

var startTime = new Date();
document.getElementById("results").innerHTML = out.join("");
alert((new Date() - startTime) + "ms");
}
</script>
</head>
<body>
Insert <input type="text" id="iterations" value="100" /> tables, using innerHTML.
<input type="button" onclick="run()" value="Execute" />
<div id="results"></div>
</body>
</html>
