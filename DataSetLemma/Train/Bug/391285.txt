Reproducible: Didn't try

Steps to Reproduce:
The following is probably not a minimal crash procedure; it is what I did just prior to the crash.
1. Load a Bugzilla bug page
2. Open DOM Inspector (Tools => Web Development => DOM Inspector)
3. Inspect the current (Bugzilla) content page in DOM inspector
4. Select the BODY element
5. In the right pane, select "CSS Style Rules"
6. Select the bottom-most style sheet at top right and "font-family: Verdana, sans-serif" at bottom right
7. Edit the rule, set value to "Bitstream Vera Sans Mono". Click OK. The font is changed in the page.
8. Edit the same rule, set to "Andale Mono". Click OK. The font is changed again.
9. Edit again. Change to "B&H LucidaTypewriter". Click OK. Nothing happens.
10. Repeat 9. Nothing happens again.
11. Try "B&amp;H LucidaTypewriter". Nothing happens
12. Try "Gnu Unifont Mono". Click OK. Crash.
Actual Results:
Crash

Expected Results:
No crash, change the font in the page
