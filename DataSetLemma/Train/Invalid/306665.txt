That 10 I assume is in PostScript points, and a I told above, 1 pixel on my
screen is 0.6547 PostScript points, so 10 PostScript point is:

10 / 0.6547 = 15.27

That mean that the default font size height (as I assume defined by the height
of the letter H) on my screen should be 15 pixels.

However on my screen the default font (on page where no CSS is used, and no
font is defined anywhere on the page - I checked many pages, including directory
listing of file on local hard disk), and the height of the letter H is 7 pixels.

Reproducible: Always

Actual Results:
On a 110 dpi display a 10 pt font is 7 pixel high (measured a the height of
the letter H).

Expected Results:
The letter H should be 15 pixel high.
