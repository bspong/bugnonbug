JavaScript: Cannot access parent's DOM via window.opener.<name> or window.opener.<id>
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9.0.1) Gecko/2008070208 Firefox/3.0.1
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9.0.1) Gecko/2008070208 Firefox/3.0.1

While window.opener.document.getElementById & window.opener.document.getElementsByTagName do work, I am not able to access (even for read only) window.opener.<name> or window.opener.<id>. Similar error occur with window.opener.document.<name> and window.opener.document.<id>. Note that I do recognize that access to the DOM via the element name or ID is not preferred, however there are several internal web-sites (that I have no control over) that use this antiquated method.

For reference, the specific error that is pulled up in the error console (presuming the element name is "theText") is "window.opener.theText is undefined" or "window.opener.document.theText is undefined".
