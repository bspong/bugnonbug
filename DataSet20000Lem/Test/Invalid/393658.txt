error after adding RSS feed to sub-folder of Bookmarks Toolbar
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9a8pre) Gecko/2007082305 Minefield/3.0a8pre
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9a8pre) Gecko/2007082305 Minefield/3.0a8pre

After adding RSS feed (Live Bookmarks) to a sub-folder of the Bookmarks Toolbar via Bookmarks Manager and exiting the application future launch produce an Assertion failure.

Reproducible: Always

Steps to Reproduce:
1. Add RSS feed to sub-folder of Bookmarks Toolbar
2. Exit application
3. Start application again



Assertion Failure:

ASSERT: null node
Stack Trace:
0:PU_nodeIsFolder(undefined)
1:oncblpopuphidden([object MouseEvent])

If the Bookmarks Toolbar is hidden, the error will pop when opening the Bookmarks Manager or History. Also ha the side-effect of disabling Forward, Back, Refresh and Stop navigation.
