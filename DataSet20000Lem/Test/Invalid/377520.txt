Tab switch when filling out form with special char using Alt-Gr
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8.1.3) Gecko/20070309 Firefox/2.0.0.3
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8.1.3) Gecko/20070309 Firefox/2.0.0.3

When filling out forms, ie. e-mail, when typing @ (Alt-Gr + 2) Firefox switch to tab #2.
When making BBCode ([] tags), Firefox switch to either tab #7 or tab #8

Only seems to happen with <input> field

Reproducible: Always

Steps to Reproduce:
1. Focus input field
2. Type name@email.com
Actual Results:
Firefox switch to tab #2, AND insert @

Expected Results:
Only the @ should be inserted, and the current tab should stay focused
