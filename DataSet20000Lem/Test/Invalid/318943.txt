Customize toolbars cause menu bar to vanish
User-Agent: Mozilla/5.0 (X11; U; Linux i686; en-US; rv:1.8) Gecko/20051202 Firefox/1.5
Build Identifier: Mozilla/5.0 (X11; U; Linux i686; en-US; rv:1.8) Gecko/20051202 Firefox/1.5

I discovered accidentally that when I right-click on the navigation bar and select "Customize", the menu bar (File, Edit, etc) disappears entirely and doe not return when the customize window is closed. The spinner doe not disappear; only the menu items.

Reproducible: Always

Steps to Reproduce:
1. Bring up the Navigation bar context menu
2. Select "Customize"
3. Close the customization window

Actual Results:
The menu bar disappears.

Expected Results:
The menu bar should not disappear.
