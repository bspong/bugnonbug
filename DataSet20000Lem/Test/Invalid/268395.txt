I request new preference setting to specify behavior when clicking on link
internal to the web site on which they appear (lacking the "http://" in the
anchor tag"), and also to specify on behavior when clicking on link external to
the web site on which they appear (including the "http://" in the anchor tag").

Example:

When clicking on a link to a page within the current web site:
o Replace current page
o Open new page in new background tab
o Open new page in new foreground tab
o Open new page in new window

When clicking on a link to a page external to the current web site:
o Replace current page
o Open new page in new background tab
o Open new page in new foreground tab
o Open new page in new window

When clicking on a link to search result in one of my search engines:
o Replace current page
o Open new page in new background tab
o Open new page in new foreground tab
o Open new page in new window
