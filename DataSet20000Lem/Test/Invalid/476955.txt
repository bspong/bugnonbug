Incompatibility with Digital Persona extension
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 6.1; en-US; rv:1.9.0.6) Gecko/2009011913 Firefox/3.0.6
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 6.1; en-US; rv:1.9.0.6) Gecko/2009011913 Firefox/3.0.6

With the extension installed, downloads of .exe (and some other) file will not start. However, if one open link in new tab or window, it will begin.

Reproducible: Always

Steps to Reproduce:
1. Digital Persona extension is installed and enabled
2. Try to download any .exe file

Actual Results:
Nothing happens.

Expected Results:
Download dialog to pop up.

I am reporting this bug from 3.0.6, but I do not have my laptop to check if this is still happening. It wa in 3.0.5 and vista ultimate x64, and also window 7 beta.

It is possible this is a dell issue - I'm using a dell studio 15 (1535). It also occurs on the slightly newer studio 1537.
