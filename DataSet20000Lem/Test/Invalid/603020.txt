[Firefox/4.0b8pre] Download Manager Pause / Resume Not Available
User-Agent: Mozilla/5.0 (Windows NT 6.0; rv:2.0b8pre) Gecko/20101008 Firefox/4.0b8pre
Build Identifier: Mozilla/5.0 (Windows NT 6.0; rv:2.0b8pre) Gecko/20101008 Firefox/4.0b8pre

The download manager doesn't allow downloads to be paused.

Reproducible: Always

Steps to Reproduce:
1. Start a download
2. Try to pause it either by right-clicking on it, or clicking the || button.
Actual Results:
The download doe not pause, cancel is the only available option.

Expected Results:
The download manager should pause the download, and allow later resuming.
