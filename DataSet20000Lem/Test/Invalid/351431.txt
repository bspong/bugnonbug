User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9a1) Gecko/20060905 Minefield/3.0a1
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9a1) Gecko/20060905 Minefield/3.0a1

When loading Windows Live at live.com, the feed all hang on "Loading...". Similarly, when pressing "Add Stuff...," the box also hang on loading. Pressing "Page 2" yield no data, even if there are supposed to be feed there.

The problem doe not occur in Firefox 1.5.0.6 nor Beta 2, but only in Minefield (3.0a1).

Reproducible: Always

Steps to Reproduce:
1. Go to www.live.com
2. Sign in to passport account
3. Wait at Windows Live Home Page

Actual Results:
All feed hang on "Loading..."

Clicking on Page 2 yield nothing.

Expected Results:
In both cases, it should load appropriate feed and information.
