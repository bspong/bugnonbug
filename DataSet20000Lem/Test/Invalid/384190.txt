"Unresponsive Script" error on window close
User-Agent: Mozilla/5.0 (Macintosh; U; Intel Mac OS X; en-US; rv:1.9a6pre) Gecko/20070611 Minefield/3.0a6pre
Build Identifier: Mozilla/5.0 (Macintosh; U; Intel Mac OS X; en-US; rv:1.9a6pre) Gecko/20070611 Minefield/3.0a6pre

When closing a window the window is destroyed, then the browser become unresponsive for fifteen seconds. The window is then re-displayed and an error dialog box is displayed over the window indicating an "Unresponsive Script." The user is prompted to Continue or Stop Script.

Reproducible: Always

Steps to Reproduce:
1. Open a window
2. Close the window
3. Wait for error message
Actual Results:
Error message is displayed after 15 second indicating an Unresponsive Script.

Expected Results:
Window closes.
