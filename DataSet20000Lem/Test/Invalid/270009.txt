with "new tab" a default for external link, opening bookmark in second window open new tab in first window
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; rv:1.7.3) Gecko/20040913 Firefox/0.10.1
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; rv:1.7.3) Gecko/20040913 Firefox/0.10.1

If you have "New Tab" chosen a the behavior for external link (using the
Tabbed Browsing Preferences extension), opening a bookmark in a second window
open a new tab in the first window. It should open a new tab in the current
window.

Reproducible: Always
Steps to Reproduce:
1. set "New Tab" a option for external link in Tabbed Browsing Options
2. start with one window that ha multiple tab open
3. open a second window
4. select a bookmark in this second window

Actual Results:
new tab for the selected bookmark is created in the first window

Expected Results:
open a new tab in the current window for the selected bookmark
