Firefox hang (becomes non-responsive) when closing a new window with a Flash applet
User-Agent: Mozilla/5.0 (compatible; Konqueror/3.5; Linux) KHTML/3.5.7 (like Gecko)
Build Identifier: Mozilla/5.0 (X11; U; Linux i686; en-US; rv:1.8.1.6) Gecko/20070208 Mandriva/2.0.0.6-2mdv2008.0 (2008.0) Firefox/2.0.0.6

Firefox hang (= becomes non-responsive) after opening a new window with a Flash applet, and closing it.


Reproducible: Always

Steps to Reproduce:
1. Make sure the Flash Plugin is Installed.
2. Access http://www.youtube.com/watch?v=WQ6hfXdxw5w .
3. Right click on a URL in the Related video box, and select "Open in a New Window".
4. Close the new window.
5. Firefox hangs.

Actual Results:
Firefox hangs.

Expected Results:
Firefox should continue to run properly.
