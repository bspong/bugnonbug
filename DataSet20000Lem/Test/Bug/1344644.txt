If Edge is running, Firefox freeze when starting the import Wizard
Created attachment 8843867
Import Wizard

[Affected versions]:
Firefox 52.0 (Build Id:20170302120751)
Firefox 53.0a2 (Build Id:20170303004003)
Firefox 54.0a1 (Build Id:20170303030202)
Firefox 52.0esr (Build Id:20170303022339)
Firefox 45.8.0esr (Build Id:20170301181722)

[Affected platforms]:
Windows 10.

[Steps to reproduce]:
1.Launch Microsoft Edge 38.14393.0.0.
2.Launch Firefox.
3.Access the about:preferences#security page.
4.Click on the “Saved Logins” button.
5.Click the “Import” button.
6.Observe that Firefox freeze and the import Wizard open after a significant delay.

[Expected result]:
The import Wizard open right after the “Import” button wa clicked.

[Actual result]:
Firefox freeze and the import Wizard open after a significant delay.
