Address bar arrow down show garbage text if bookmark are empty
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9b5) Gecko/2008032620 Firefox/3.0b5
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9b5) Gecko/2008032620 Firefox/3.0b5

If I open the bookmark Library (Ctrl+Shift+B) and I delete all bookmark (except the default folder with "Delete" option disabled), then the arrow down at the right of the address bar show garbage text instead of the recent site or link I visited.

Reproducible: Always

Steps to Reproduce:
1. Run installer and install the program (v3beta5) with all the default options.
2. Go to Tools->Options->Main and set show a blank page at startup.
3. Go to Tools->Options->Privacy and disable the "Keep history..." option.
4. Open the bookmark library by clicking Ctrl+Shift+B
5. Delete all bookmark that firefox allow to delete.
6. Clear all private data.
7. Restart firefox.
8. Start navigation (google.com, make a search, click on results, ect).
Actual Results:
The arrow down at the right of the address bar (address bar history) show 3 line of garbage text when I click on it.

Expected Results:
The arrow down should show what I have entered in the address bar or link I clicked.
