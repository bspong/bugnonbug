Javascript us java method of applet - only work with applet tag not with embed tag
User-Agent: Mozilla/5.0 (X11; U; Linux i686; en-US; rv:1.8.0.12) Gecko/20080418 CentOS/1.5.0.12-15.el5.centos Firefox/1.5.0.12
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; de; rv:1.8.1.14) Gecko/20080404 Firefox/2.0.0.14

I try to call a method of a java applet from javascript.
If i declare the applet in the HMTL file with an <APPLET> Tag it work fine. Declaring it with an <EMBED> Tag cause the call to the applet method not to function. That mean these method are never called.

You can verify this by using http://www.kornfellner.com/java/mozbug-ok.html (working, using applet tag) and http://www.kornfellner.com/java/mozbug-nok.html (not working using embed tag).

Reproducible: Always

Steps to Reproduce:
1. goto http://www.kornfellner.com/java/mozbug-nok.html
2. click the link on the page - nothing will happen
3. goto http://www.kornfellner.com/java/mozbug-ok.html
4. click the link on the page - the label in the applet will change
Actual Results:
mozbug-nok.html -> nothing
mozbug-ok.html -> label in applet change

Expected Results:
on both html page the label in the applet should change if you click the link

I tried it with Firefox 3 beta 5 (Mozilla/5.0 (Windows; U; Windows NT 5.1; de; rv:1.9b5) Gecko/2008032620 Firefox/3.0b5) and it show the same behavior.
