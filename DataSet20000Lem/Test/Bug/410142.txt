Affecting multiple version of Firefox on WinXP & Vista. Not dependant on websites, nor on usage load.

Intermittently, Firefox will disappear from the screen (almost a though it ha been minimised), but is not visible in the Taskbar. Opening the Task Manager show the firefox process continuing to run, a doe hitting Alt-Tab until the Firefox application is selected, at which time it return to the screen.

This give the user the impression that Firefox ha crash silently, when it ha not so much crashed, but ha instead disappeared.

There is no obvious pattern which will replicate this issue.
Fault ha been experienced on different version of Firefox, including the General Firefox release, FrontMotion's version of Firefox and Firefox Portable.
Has also affected Firefox running without any Extensions, a well a Firefox loaded with >20 Extensions.

As above -
Intermittently, Firefox will disappear from the screen (almost a though it ha been minimised), but is not visible in the Taskbar. Opening the Task Manager show the firefox process continuing to run, a doe hitting Alt-Tab until the Firefox application is selected, at which time it return to the screen.
