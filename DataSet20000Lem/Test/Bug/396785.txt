Maxi Vista multiple monitor software remote clip board copy feature and clipboard synchronization feature doe not work after upgrading to Firefox 2.0.0.7. It work fine using IE.

Reproducible: Always

Steps to Reproduce:
1. Load Maxi Vista
2. Set up Hot Keys in Remote PC.
3. Copy something to clip board.
4. Press Hot Key to Copy remote clipboard.
5. Nothing happens.
Actual Results:
Noting

Expected Results:
The content of the remote clipboard should appear in the clip board of the main computer.
