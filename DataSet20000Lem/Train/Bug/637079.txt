It is impossible to edit an existing Facebook note under Firefox. After clicking the Edit button you can mark the text but not change it in any way. It is still possible to create new note though. Problem occurs only when editing an existing note.
Same thing work fine with IE8. Didn't test Chrome.

Here is a thread of affected Facebook users: http://www.google.co.il/url?sa=t&source=web&cd=1&ved=0CBQQFjAA&url=http%3A%2F%2Fwww.facebook.com%2Ftopic.php%3Fuid%3D2309869772%26topic%3D19947%26post%3D120094%26_fb_noscript%3D1&rct=j&q=facebook%20notes%20edit%20firefox&ei=7IVpTbCHCcy0hAfgk4DtDg&usg=AFQjCNFymtrwyws7GuvOI-zTxlbc6jjQcw&cad=rja

Reproducible: Always

Steps to Reproduce:
1. Log in to Facebook.
2. Create and post a note.
3. Try to edit that note in Firefox - doesn't work.
Actual Results:
Can't edit text of note; possible only to mark text.

Expected Results:
Should be possible to edit and safe the text.

IE8 work fine. Didn't test Google Chrome.
