User-Agent:
Build Identifier: Mozilla/5.0 (X11; U; Linux ppc; en-US; rv:1.6) Gecko/20040209 Firefox/0.8

New Themes" the browser close unexpectedly after I click the ok button. It
make no difference whether or not I check the box that say "use this theme".
I am running gentoo linux on a powerpc system, so I had to compile the browser
from source. This maybe a problem on my end, but I thought I would give you
people head up.

Reproducible: Always
Steps to Reproduce:
1.select the Options... menu item from the Tools menu
2.click the theme button
3.click the "get new themes" hyperlink
4.select a theme from the webpage to download
5.hit the ok button when the dialog pop up.

Actual Results:
All browser window close

Expected Results:
It should have properly installed the theme and kept all window open

using the default theme
