In Firefox, the YouTube search box doesn't always display the query suggestion
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-GB; rv:1.9.0.5) Gecko/2008120122 Firefox/3.0.5
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-GB; rv:1.9.0.5) Gecko/2008120122 Firefox/3.0.5

The search box from YouTube doesn't display the query suggestion in the video page (It doe display the query suggestion in the YouTube home page and video lists).
Note: This work flawlessly in Internet Explorer.

Reproducible: Always

Steps to Reproduce:
1. Open YouTube home page.
2. Click the "Search" button without anything written in the search box. A link with "Advanced Options" shuld appear under the "Search" button.
3. Click on "Advanced Options" and make sure that "Videos" is highlighted.
4. Check "Display query suggestion a I type" and click the small "Close" button from the upper right corner.
5. Go to the YouTube's Home page and play any video you find there.
6. In the page with the video playing, write any letter in the "Search" box.
Actual Results:
The drop-down list with suggestion fails to appear.

Expected Results:
The suggestion drop-down list should appear, displaying search suggestion starting with the letter or letter that you wrote in the "Search" box.

This problem also occured in Mozilla Firefox 2.
This problem doesn't occur in Internet Explorer 7.
