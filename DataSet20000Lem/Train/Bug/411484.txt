Middle-click folder on toolbar open content in tabs, but Cmd+click/Ctrl+click doesn't
User-Agent: Mozilla/5.0 (Macintosh; U; Intel Mac OS X 10.5; en-US; rv:1.9b2) Gecko/2007121014 Firefox/3.0b2
Build Identifier: Mozilla/5.0 (Macintosh; U; Intel Mac OS X 10.5; en-US; rv:1.9b2) Gecko/2007121014 Firefox/3.0b2

COMMAND+click item on the Bookmarks Toolbar open that item in a new tab, middle-click the same item also open the item in a new tab.

COMMAND+click bookmark folder on the Bookmarks Toolbar expose the folder's content (similar to left-click). Middle-click the same folder open the folder's content in new tabs.

These should be reconciled.

Reproducible: Always

Steps to Reproduce:
1. Create folder of bookmark on the Bookmarks Toolbar and populate with > 1 bookmark
2. Command-click on this bookmark folder
3. Menu item display folder's content
4. Middle-click on this bookmark folder
5. Folder's content open in new tab

Actual Results:
Command-click display folder's content in a menu item, middle-click open folder content in a new tab

Expected Results:
Both action should result in opening folder content in new tabs, similar to result of these action on an individual bookmark item on the toolbar
