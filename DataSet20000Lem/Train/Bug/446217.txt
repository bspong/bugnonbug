Clear Private Data appears when Firefox 3.0.1 start after upgrade but not when it close
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9.0.1) Gecko/2008070208 Firefox/3.0.1
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9.0.1) Gecko/2008070208 Firefox/3.0.1

I just upgraded my Firefox browser from v2.0.0.15 to v3.0.1.

The Clear Private Data appears when Firefox 3.0.1 start after upgrade but not when it closes.

Workaround wa to untick the option from "Tools -> Option -> Privacy tab -> Always clear my private data when I close Firefox" and then close the browser.

Relaunch the browser and reverse the above action, that is enable clearing of private data, this then enables the "Clear private data" window to appear after closing the browser.
