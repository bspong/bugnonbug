Address bar and search bar can not be be selected after logging in to WebCT/Blackboard.

After logging in to my school course management website I can no longer select the address bar or search bar or enter any text on the webpage. Restarting the browser brings it back to normal. I never had this trouble before upgrading to this newest version of Firefox.

Reproducible: Always

Steps to Reproduce:
1. Log in to a WebCT/Banweb server (https://courses.mtu.edu)
2. Dialog box pop up: "Do you want to allow courses.mtu.edu access to your computer?" Answer "Allow"
3. Attempt to select the address bar or search box and enter text.
Actual Results:
No cursor show up, and text can not be entered into any text field, including the address bar and search bar.

Expected Results:
Expect to see a cursor, and to be able to enter text.

The WebCT server I logged in to to initiate this claim to be the following version:
Blackboard Learning System - CE Enterprise License (Release CE 8.0.2) [18.0.2.42]

The following add-on are installed and at the newest version:
Adblock Plus 1.0.2
Add-Art 0.8.3
