On "Bookmark This Page ..." the "Show all the bookmark folders" and the "Add" button don't work
User-Agent: Mozilla/5.0 (Macintosh; U; PPC Mac OS X Mach-O; en-US; rv:1.8b4) Gecko/20050825 Firefox/1.0+
Build Identifier: Mozilla/5.0 (Macintosh; U; PPC Mac OS X Mach-O; en-US; rv:1.8b4) Gecko/20050825 Firefox/1.0+

When you want to bookmark a page and click on "Bookmarks"/"Bookmark This Page
..." oder type Cmd+D the "Show all the bookmark folders" and the "Add" button
don't work a expected. On clicking "Add" nothing happens, on clicking "Show all
the bookmark folders" it flip to "Hide" and the "New Folder..." button appears
but the listing of the bookmark folder doesn't appear.

Reproducible: Always

Steps to Reproduce:
1. click on "Bookmarks"/"Bookmark This Page ..." or type Cmd+D
2. click on "Add" or "Show all the bookmark folders"
3.

Actual Results:
On clicking "Add" nothing happens, on clicking "Show all the bookmark folders"
it flip to "Hide" and the "New Folder..." button appears but the listing of the
bookmark folder doesn't appear.

Expected Results:
Clicking on "Add" should add the currently loaded page a bookmark. Clicking on
"Show all the bookmark folders" it should flip to "Hide", the "New Folder..."
button should appear a well a the listing of all the bookmark folders.
