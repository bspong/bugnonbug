When logged in to a site with username/password authentication,with firefox option set to remember password.The next time when you login to that site, the password field will be filled with black dot and user name field is empty. When the username is typed and submitted the black dot vanished and say empty password. You have to type in the password again to work...

Reproducible: Always

Steps to Reproduce:
1.Log in to any site having a username/password authentication
2.When logged in firefox will ask if you want to remember password for that site- answer YES
3.Close the tab and Restart firefox (Optional it seems) and try logging in again
Actual Results:
The password field is filled with black dot and username field is empty!!. When the username is typed and submitted the black dot vanished and say empty password. You have to type in the password again to work...

Expected Results:
Both username and password field should be seen a filled with the value given to remember.
