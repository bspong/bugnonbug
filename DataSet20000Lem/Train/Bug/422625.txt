Reproducible: Always

Steps to Reproduce:
1. Log in to Google Analytics (assuming pre-registration)
2. Select Dashboard option (or any option that provides access to date range comparison)
3. Select month name at top, eg. March

Actual Results:
Browser hang momentarily, and a crash report is produced

Expected Results:
Date range becomes highlighted within the screen, and cursor move focus to next input to allow selection of comparable date range.
