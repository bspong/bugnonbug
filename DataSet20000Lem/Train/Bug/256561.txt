clicking on a link in an external application fails to launch firefox
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.7) Gecko/20040803 Firefox/0.9.3
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.7) Gecko/20040803 Firefox/0.9.3

When any other application (for example: mozilla thunderbird mail messages) have
embedded link to urls, clicking on them fails to launch a browser instance,
unless a browser is already open. If one is open, then the embedded link work
perfectly.

Reproducible: Always
Steps to Reproduce:
1. open external application with embedded url without firefox running
2. click on it
3. nothing happens

Actual Results:
nothing happens

Expected Results:
launched firefox browser and opened webpage referenced by url
