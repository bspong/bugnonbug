When an RSS Feed is in the overflow region on the bookmark toolbar the Folder icon display instead of the RSS Feed icon. Reproduced on 2 OSX systems.
Seems to not be reproduced on window based on feedback in IRC channel.

Reproducible: Always

Steps to Reproduce:
1. Add some RSS feed to the bookmark toolbar
2. Resize you browser width so that an RSS feed is off the Browser display.
3. Click on the >> to view the RSS Feeds on the overflow region.
4. Check that the icon are correct

Actual Results:
RSS Feed display with Folder icon instead of RSS Feed icon.

Expected Results:
RSS Feed icon is displayed for each RSS Feed.
