Opening several link from Outlook 2003, then closing first window, cause links/address bar to stop working
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.7.8) Gecko/20050511 Firefox/1.0.4
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.7.8) Gecko/20050511 Firefox/1.0.4

Using Microsoft Outlook 2003, with Firefox (v1.4) initially closed:
When viewing an email with several link and then clicking on those links,
Firefox open several new windows. Initially, everything work fine. However,
if I then close the first window opened (the one corresponding to the first link
clicked in the email), the link in the remaining window stop working (i.e.
nothing happens if you click on them). Furthermore, the address bar also stop
working (i.e. typing, for instance, www.google.com, and hitting enter doe nothing).

Reproducible: Always

Steps to Reproduce:
1. Make sure Firefox is not running
2. open (any) email with several link in Microsoft Outlook
3. Click on those links, opening several new browser window
4. Close the first of those new window that opened (the one corresponding to
the first link clicked in the email)
5. Click on any link in any of the other windows, or type a new address in the
address bar and hit enter

Actual Results:
Nothing

Expected Results:
Go to the page of the link clicked or the address entered
