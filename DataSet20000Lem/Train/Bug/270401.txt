Desktops shortcut for URLs are opened always in the same window of the browser, they should be opened in a blank and separate browser window
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.2; en-US; rv:1.7.5) Gecko/20041107 Firefox/1.0
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.2; en-US; rv:1.7.5) Gecko/20041107 Firefox/1.0

Desktop shortcut for URLs are opened always in the same window of the browser,
when they should be opened in different window of the default browser. There is
no way to configure this. This doe not happens with other browsers. When I load
a page, then I go to my desktop and want to open another URL, Firefox open in
the one I am surfing. This is not the way it work with other browsers, clicking
in other URL shortcut it make the page open in a separate or blank browser window

Reproducible: Always
Steps to Reproduce:
1. Save some URL shortcut for different web sites, this can be done in a folder
or in Windows desktop
2. Click in one of them to open the website
3. Click in another shortcut saved in step 1.
4. The website for step 3 it will be opened in the same browser than the first
one opened in step 2

Actual Results:
Desktop shortcut for different website URLs are opened always in the same
window of the browser

Expected Results:
Desktop shortcut for different website URLs it must be opened always in other
blank window of the default Windows browser
