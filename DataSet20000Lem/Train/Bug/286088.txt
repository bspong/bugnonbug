After client upgrade to latest Firefox version (1.01), browser hog system resource repeatedly for overall slow PC performance
User-Agent: Mozilla/4.0 (compatible; MSIE 6.0; Windows NT 5.1; SV1; .NET CLR 1.1.4322)
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.7.6) Gecko/20050225 Firefox/1.0.1

Every few second after starting and using the browser to go to a variety of
Web sites, the system resource used on the PC peak to 100% and affect not
only use of the browser, but use of other application while system resource
are being hogged by Firefox.

Reproducible: Always

Steps to Reproduce:
1. Start Firefox client software (v. 1.01)
2. Try to browse any number of Web site
3. Every few seconds, usage of system resource peak at 100% and slows
EVERYTHING down.

Actual Results:
Every resource on my PC slows down since Firefox is hogging all available
system resource every few seconds.

Expected Results:
It should NOT be a consistent resource hog, and allow quick and easy access
from within the browser, a well a from other application running on the PC.

The PC system on which the bug is occuring is a Dell Dimension 8400 with a
3.40GHz processor, 4.0 GB of memory and an ATI Radeon X800 XT video card.
