Firefox v1.5 drop down menu do not change after changing from right handed to left handed
User-Agent: Mozilla/4.0 (compatible; MSIE 7.0b; Windows NT 5.1; .NET CLR 1.1.4322; .NET CLR 2.0.50727)
Build Identifier: Firefox v1.5 from http://www.mozilla.com

THIS PROBLEM IS WITH FIREFOX USING TABLETPC COMPONETS ON VISTA BUILD 5342.

Dropdown menu are the same for right-handed and left-handed settings.


Reproducible: Always

Steps to Reproduce:
**Environment Motion LE1600**
SKU: Home Premium

1) install 5342.vbl_wcp_mobile
2) install Firefox v1.5 from http://www.mozilla.com
3) Open Firefox
4) Select toolbar item to view dropdowns
5) Close Firefox
6) Open Tablet PC Settings (Control Panel > Mobile PC > Tablet PC settings)
7) Change from Right-Handed to Left-Handed and click Apply
8) Open Firefox
9) Select toolbar item to view dropdowns


Actual Results:
The menu drop to the right no matter which hand it is set to.

Expected Results:
When the right-handed setting is enabled the menu should drop to the left, when the left-handed setting is enabled the menu should drop to the right


Should open on the left or right depending on handedness wa set.
