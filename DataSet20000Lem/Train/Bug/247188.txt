After pressing Enter in search bar, dragging to search bar open a new tab
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.0; en-US; rv:1.7) Gecko/20040614 Firefox/0.9
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.0; en-US; rv:1.7) Gecko/20040614 Firefox/0.9

When I first perform search on a text selection dragged to the search bar, the
result (from Google) are displayed in the same browser window. However, after
doing a Find In This Page search, subsequent drag and drop web search result
are returned in new tabs, though I didn't change any preference settings.

Once I exit and restart the browser, the searchbar return to it default
behavior of returning web search result in the same browser window.

Reproducible: Always
Steps to Reproduce:
1. Load a page in the browser, select a word, drag the selection to the search
bar, and observe that the result are returned in the same browser window.
2. Select the "Find In This Page" option from the search bar drop down menu
list, type or drag some text to the bar, and hit enter to perform the Find In
Page Search.
3. Select "Google" engine from search bar drop down menu list. Then select a
word and drag it to the search bar a in step 1. The result of the google
search are returned in a new tab.

Actual Results:
Web search result were returned in new tabs.

Expected Results:
Returned web search result in the same window.
