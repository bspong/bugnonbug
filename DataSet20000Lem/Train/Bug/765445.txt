X-Tag demo don't work on Arch Linux build
Created attachment 633766
Everything from about:support

Most of the demo at http://mozilla.github.com/x-tag/demo.html do not work in the Arch Linux build of Firefox. The Mozilla build of Firefox ha no problem on them.

Addons don't seem to be the issue: I tested it both in a clean Firefox profile and in safe mode.


Demo-by-demo result
--------------------

Panel:
Works: Nothing
Doesn't work: Add src to panel, Switch src

Tab Box:
Works: Unstyle/Style, Tabs to Top/Left/Right/Bottom, Add Tab
Doesn't work: Select Previous/Next Tab, clicking on tab

Accordion:
Works: Unstyle/Style, Add Section
Doesn't work: Select Previous/Next Section, clicking on section

Slide Box:
Works: Nothing
Doesn't work: Previous/Next Slide, Slide to 3, Toggle orientation

Flip Box:
Works: Switch Flip Direction (toggles between up/down/right/left)
Doesn't work: Flip

Modal:
Works: Show, Unstyle/Style, Toggle Overlay, Close
Doesn't work: Nothing

Map:
Works: Nothing
Doesn't work: The map doesn't show up

Autosuggest:
Works: Nothing
Doesn't work: The text box doesn't show up

Pager:
Works: Nothing
Doesn't work: The link don't show up

Action Bar:
Works: Nothing
Doesn't work: The button don't show up
