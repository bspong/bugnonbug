I run a wiki and I use firefox because I can do work with it. It support masive tabbed browsing, and it used to support precise clicking.

I recently downloaded the new version of firefox and noticed that now, when I am deleting blank space in a wiki-template, for example, it sometimes auto-deletes entire parameter or word when I only want to delete one blank space. Basicially the new version of firefox, in gamer terms, ha decreased the ability to micro.

The auto-deletion really need to be removed, it make it a lot harder to work with.
