First item in Bookmarks subfolder show Empty
Description:

Test environment:
Windows XP, Firefox 2 (Bon Echo) build 20060731, Microsoft Inspect Objects

Steps to recreate:
1. Make sure you have at least one bookmark saved in a subfolder in the bookmark pulldown. If you do not have a subfolder in your bookmark pulldown, create one like this:
a. Open a Web page in Firefox
b. Press Ctrl+D to open Add Bookmark dialog.
c. Click on the "down arrow" button to the right of the "Create in:" field.
d. Select "New Folder" button at bottom of the window.
e. Select OK in all resulting dialog to complete the Add Bookmark feature. This should create a subfolder in the bookmark menu.

2. Display the content of the subfolder in the Bookmarks pulldown, using these instructions:
a. Press Alt+B to open bookmark menu pulldown
b. Press Down Arrow key until you have highlighted the subfolder in the bookmark folder (created above in step 1).
c. Press Right Arrow to open the subfolder.

3. Note the first item in the subfolder list show (Empty). Directly underneath the word (Empty) you see the actual content of the subfolder.
4. If you close the subfolder (Press Left Arrow) and reopen it (Press Right Arrow), the (Empty) item disappears, and the subfolder is displayed a expected.

Expected results:
The first item in the bookmark subfolder should not say (Empty).
