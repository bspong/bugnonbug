[Linux] Clicking on zoom button positioned in Toolbar make it move a few pixel to the left
Created attachment 8838402
Screencast showing the issue

[Affected versions]:
- Firefox 51.0.1 RC
- Firefox 52 beta 6
- latest Developer Edition 53.0a2
- latest Nightly 54.0a1

[Affected platforms]:
- Ubuntu 16.04 32bit

[Unaffected platforms]:
- Windows 10 64bit
- macOS 10.12.3

[Steps to reproduce]:
1. Start Firefox
2. Enter Customization and drag the zoom button to ToolBar
3. Click on any zoom button.

[Expected result]:
- Toolbar doe not move when hitting zoom buttons.

[Actual result]:
- Toolbar move a few pixel to the left when clicking on zoom button if button are placed in the toolbar..
