url shortcut doesn't work
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.0; en-US; rv:1.7.5) Gecko/20041107 Firefox/1.0
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.0; en-US; rv:1.7.5) Gecko/20041107 Firefox/1.0

When you create a Windows URL shortcut, the browser can't load it.

Reproducible: Always
Steps to Reproduce:
1. Right click on the desktop.
2. Click 'New'.
3. Click 'Shortcut'.
4. A dialog box entitled "Create Shortcut" will appear.
5. In the text box entitled "Type the location of the item:" type
"http://www.mozilla.org/products/firefox/".
6. Click the "Next" button.
7. In the text box entitled "Type a name for this shortcut:" type "Firefox".
8. Click the "Finish" button.
9. Make sure there are no browser open, and that Firefox is your default browser.
10. Double click the Firefox shortcut icon on the desktop.

Actual Results:
Firefox open with no page loaded. A popup dialog appears that state that the
URL can't be found.

Expected Results:
Firefox should have successfully loaded the shortcut URL.

Under "Severity" you ought to have a category for "Problem bad enough to cause
you to use a different product" between Major and Critical. Because of this
problem, I will need to switch back to Internet Explorer. If I can get it to be
my default browser again...
