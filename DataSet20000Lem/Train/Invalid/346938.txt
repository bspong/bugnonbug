If I open Safari... Preferences and select Deer Park a the Default Browser OR set it a the Default Browser in Deer Park Preferences, and then close the browsers. If I double-click a Link or URL file, it will most likely open Firefox, not Deer Park, and I need to change the Default Browser again. Very annoying. Unless I already have Deer Park open.

Reproducible: Always

Steps to Reproduce:
1. Set Default Browser in Deer Park or Safari.
2. Close Browser
3. Double-click URL file or link in another application.

Actual Results:
Firefox opens, not Deer Park.

Expected Results:
Deer Park would open, not Firefox.
