Build Identifier: Mozilla/5.0 (Macintosh; U; PPC Mac OS X Mach-O; en-US; rv:1.9a5pre) Gecko/20070601 Minefield/3.0a5pre

Trying to move multiple item (bookmarks or folders) between folder by drag-and-drop, I noticed that only one item is moved. Rest of the selection stay in the source folder

Reproducible: Always

Steps to Reproduce:
1. Open Firefox (Minefield version mentioned in the bug)
2. Open 'Bookmarks -> Organize Bookmarks...'
3. Select multiple item (folders or bookmarks)
4. Drag the selected item to a different folder
Actual Results:
Out of all the selected items, only one item is moved. Rest stay put.

Expected Results:
All selected item should be moved
