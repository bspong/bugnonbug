My eBay - dropdown Action list no longer work
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8.0.1) Gecko/20060111 Firefox/1.5.0.1
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8.0.1) Gecko/20060111 Firefox/1.5.0.1

If you are watching item on eBay, the dropdown Action list in the "Items I'm Watching" section no longer seem to work. The Action area consists of a "Bid Now" or "Buy It Now" button and a dropdown menu icon. Selecting the dropdown menu icon should bring up a list of appropriate actions. Looking at the source, this appears to be done using Javascript. Please note, the same problem applies to all other dropdown Action menu in other section of the My eBay page.

Reproducible: Always

Steps to Reproduce:
1. login to eBay
2. Go to My eBay
3. Hit the dropdown Action menu for an item you are watching

Actual Results:
Nothing happens

Expected Results:
A list of possible action should appear

Applies to all dropdown Action menu in all section of the My eBay page.
