Firefox forgets all setting after restart. I mean
- startup & home page
- if it should be the default browser and if it should check
- other similar setting on which Firefox asks on startup / shutdown
- theme

Reproducible: Always

Steps to Reproduce:
1.I just updated from Firefox 3.5 (by Help/check for updates).
2.After update & restart to the new version I wa asked if it should be my default browser and if it should check. I responded no default, no check.
I noticed that my homepage wa changed to http://en-US.start3.mozilla.com/firefox?client=firefox-a&rls=org.mozilla:en-US:official and startup page from <blank> to <home page>. It used default theme, I wa using aero fox previously.
I corrected these setting to my own home page and <blank page> on default.
3. I closed Firefox.
4. I started Firefox again.
5. I noticed everything changed to state a written in step 3 (default browser question included).
I repeated step 2 - 5 about 3 times, nothing helped.

Actual Results:
Firefox forgets all setting after restart.

Expected Results:
Firefox should remember setting and show error message when not possible (to save or use).

I use win XP SP3 x86.
