Tools-Options dialog doe not save it's size.
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8a2) Gecko/20040528 Firefox/0.8
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8a2) Gecko/20040528 Firefox/0.8

Every time you go to the tools-options dialog, you must re-size the dialog. The
default size is not wide enough not to have to horizontally scroll in some of
the option windows.

Reproducible: Always
Steps to Reproduce:
1. Goto Tools->Options, resize the dialog, Click Ok.
2. Goto Tools->Options, dialog size is back to default.

Actual Results:
Dialog size back to default.

Expected Results:
Dialog should retain new size.

Running under Windows XP.
