Page setup should be easily accessible from the Print dialog
User-Agent:
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.7a) Gecko/20040128 Firebird/0.8.0+

It would be handy to bring up the Print dialog and have Page setup easily
accessible from within the dialog. Print preview currently doe this, but not
the plain vanilla print dialog.

Reproducible: Always
Steps to Reproduce:
1. Hit the print button on the toolbar.
2. Want to change page setup.

Actual Results:
Have to close the dialog and go to File, Page Setup.

Expected Results:
"Page Setup" button on the Print dialog brings up the Page Setup dialog.
