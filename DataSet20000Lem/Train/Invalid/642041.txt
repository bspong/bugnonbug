Private browsing not working a intended when started from taskbar
User-Agent: Mozilla/5.0 (Windows NT 6.1; WOW64; rv:2.0) Gecko/20100101 Firefox/4.0
Build Identifier: Mozilla/5.0 (Windows NT 6.1; WOW64; rv:2.0) Gecko/20100101 Firefox/4.0

When entering private browsing from the taskbar and performing a search from the address bar, when the result are displayed google profile will still be signed in, even if in private mode.

Reproducible: Always

Steps to Reproduce:
1. Start browser in private mode
2. Go to google.co.uk
3. Log in any google account
4. Close browser
5. Right click on the Firefox icon pinned on the taskbar and select Enter Private Browsing
6. Perform any search from the address bar
7. Wait for search result to show
8. Notice log in detail in top right corner of the google page
Actual Results:
In the search result the google profile stay signed in even if the user ha connected in private mode

Expected Results:
User shouldn't stay logged in
