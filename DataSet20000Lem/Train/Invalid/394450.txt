Delete button in Bookmark This Page dialog is misleading

Build Identifier: Mozilla/5.0 (Macintosh; U; PPC Mac OS X Mach-O; en-US; rv:1.9a8pre) Gecko/2007083004 Minefield/3.0a8pre

The dialog that come up after selecting Bookmark This Page command from Bookmarks menu contains 2 buttons. One is labeled "Delete" and the other "Done."

The Delete button should be labeled "Cancel".

Reproducible: Always

Steps to Reproduce:
1. Go to any web page.
2. Select "Bookmark This Page..." command from Bookmarks menu.
Actual Results:
Add bookmark dialog show up with a Delete button

Expected Results:
The button should be labeled "Cancel"
