"use current page" button to set homepage doesnt work properly when tab are open
User-Agent: Mozilla/5.0 (X11; U; Linux x86_64; en-US; rv:1.7.10) Gecko/20050907 Firefox/1.0.6
Build Identifier: Mozilla/5.0 (X11; U; Linux x86_64; en-US; rv:1.7.10) Gecko/20050907 Firefox/1.0.6

If you have multiple tab open and go to the prefrence pane and click "Use
Current Page" to set the homepage, it set it to the url of the left-most tab,
not the current tab.

Reproducible: Always

Steps to Reproduce:
1.Open multiple tabs.
2.Switch to tab other than the left-most one.
3.Go to Preferences and click "Use Current Page" in the General prefrences pane.

Actual Results:
Url entered into homepage location field is that of the left-most tab, not the
current tab.

Expected Results:
The url of the current tab should have been placed home page location field.
