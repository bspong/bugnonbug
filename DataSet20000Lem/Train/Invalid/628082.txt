Upon leaving PB mode, selected search engine is retained, whilst the search query wa restored from pre-PB

Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 6.1; en-US; rv:1.9.2.13) Gecko/20101203 Firefox/3.6.13 ( .NET CLR 3.5.30729; .NET4.0C)

When leaving Private Browsing mode, the search engine last used is retained a the currently selected search engine. The search query is restored from pre-PB Mode (as per bug 463202).

Reproducible: Always

Steps to Reproduce:
<example, "foo", "bar", Google, and Wikipedia can be replaced>
1. Search "foo" using Google.
2. Enter Private Browsing mode.
3. Search "bar" using Wikipedia.
4. Leave PB mode.
Actual Results:
The data in the search box wa "foo" using Wikipedia.

Expected Results:
The data should be "foo" using Google.

The bug can be replicated using any search term or providers, not just "foo" on Google and "bar" on Wikipedia.
