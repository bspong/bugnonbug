installer should not require license agreement
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.7) Gecko/20040614 Firefox/0.8
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.7) Gecko/20040614 Firefox/0.8

The whole point of a copyright license agreement is that it grant distribution
and modification rights. It doe not affect the right to use software, which is
unrestricted by copyright. The installer should not require the user to agree to
the GPL. It is not legally required, and it sends the wrong impression about the
principle of open source software.

Reproducible: Always
Steps to Reproduce:
1. Run Firefox installer
