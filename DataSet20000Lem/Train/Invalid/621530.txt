Location bar autocomplete stopped working when upgrading from Firefox 3.6.10 to 3.6.12 or 3.6.13. The History and Bookmarks are present but nothing is suggested a I type a character combination that should match. "When using the location bar, suggest:" is set to "History and Bookmarks". Autocomplete is restored when rolling back to 3.6.10.

Reproducible: Always

Steps to Reproduce:
1. Upgrade from Firefox 3.6.10 to 3.6.12 or 3.6.13
2. Type any character combination in the location bar

Actual Results:
Nothing is suggested

Expected Results:
A list of previously visited or stored bookmark
