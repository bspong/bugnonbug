Firefox doe not direct cursor to Location Bar when opening new tab
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.0; en-US; rv:1.9a8) Gecko/2007091216 GranParadiso/3.0a8
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.0; en-US; rv:1.9a8) Gecko/2007091216 GranParadiso/3.0a8

In earlier version of Firefox, you could open a tab and immediately start typing in the address bar. In Firefox 3.0a8, opening a new tab doe not auomatically direct your cursor there.

Reproducible: Always

Steps to Reproduce:
1. Create a new tab.
2. Type.
Actual Results:
Typing doe nothing.

Expected Results:
The cursor appears in the Location Bar and the user can immediately start typing the URL without having to click there.
