Copy/pasting cell with key shortcut work only once in Google Docs Spreadsheet. The first time it work fine, when trying again it copy again the first cell.

When using google doc menu instead of the keyboard: Edit - copy and Edit - paste, it work fine.

Reproducible: Always

Steps to Reproduce:
1.go to docs.google.com, create a new spreadsheet
2.write data two different cell
3.copy one cell, paste it into an empty cell
4.copy the second cell, paste it into another empty cell

Actual Results:
the content of the first cell that wa copied is pasted on the empty cell

Expected Results:
the content of the second cell that wa copied should be pasted on the empty cell
