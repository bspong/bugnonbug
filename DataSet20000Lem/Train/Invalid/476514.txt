In private mode, zoom in new tab are always 100% regardless of previous tab

Build Identifier: Mozilla/5.0 (Macintosh; U; Intel Mac OS X 10.5; sv-SE; rv:1.9.1b2) Gecko/20081201 Firefox/3.1b2

When opening a link in a new tab, the new tab ha the same zoom level a the tab it wa opened from.
However, in private mode, the new tab have always the default zoom (100%).

Reproducible: Always

Steps to Reproduce:
1. Go to e.g. http://www.pacificnanotech.com/gallery.html
2. Zoom in to e.g. 200%
3. Open one of those gallery link in a new tab
In normal mode, the new tab is 200%
In private mode, the new tab is 100%
Actual Results:
In private mode, link in new tab have always the default zoom

Expected Results:
In private mode, link in new tab should have the same zoom level a the site it wa opened from
