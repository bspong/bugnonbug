"Select new tab opened from links" pref setting not rememberred by browser
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8.0.1) Gecko/20060111 Firefox/1.5.0.1
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8.0.1) Gecko/20060111 Firefox/1.5.0.1

If you unset "Select new tab opened from links" pref option - shutdown all firefox instance and restart. Firefox re-enables this option. i.e. It doe not remember that it wa unset in the last session

Reproducible: Always

Steps to Reproduce:
1. unset the option
2. shutdown all instance of firefox
3. restart firefox
4. Check option setting again

Actual Results:
option is reset

Expected Results:
option should remain unset

none
