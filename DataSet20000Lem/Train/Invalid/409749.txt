Setting home page in preference with multiple tab open default current page to first tab, not open tab
User-Agent: Mozilla/5.0 (Macintosh; U; Intel Mac OS X 10.4; en-US; rv:1.9b2) Gecko/2007121014 Firefox/3.0b2
Build Identifier: Mozilla/5.0 (Macintosh; U; Intel Mac OS X 10.4; en-US; rv:1.9b2) Gecko/2007121014 Firefox/3.0b2

When setting the home page to the current page in preferences, the "current page" option us the first tab on the active window, not the tab in the foreground a would be expected.

Reproducible: Always

Steps to Reproduce:
1. Open two tab two different urls.
2. Open tab two in the foreground.
3. Open preferences, and select "current page" a the option for setting the home page.

Actual Results:
Sets background (first) tab, not foreground (second) tab.

Expected Results:
Sets foreground (second) tab.
