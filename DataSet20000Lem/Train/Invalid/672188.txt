[10.7] Webconsole in window mode isn't displayed in Mission Control
Mozilla/5.0 (Macintosh; Intel Mac OS X 10.7; rv:6:0) Gecko/20100101 Firefox/6.0

If user selects Mission Control while Webconsole is popped-out, it's window is not displayed (Scratchpad for example is displayed)

Steps to reproduce:
1. Open Webconsole
2. From Position Menu choose Window
3. Display Mission Control

Actual results:
- the webconsole window is not displayed

Expected results:
- the webconsole window is displayed and the user can select it.
