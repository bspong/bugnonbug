On Tools > Options > Content > Fonts & Colors, and also the Advanced... submenu for Fonts and Colors, on WinXP the "Size: [...]" pull down list of font size lack horizontal width to show the current font size.

Reproducible: Always

Steps to Reproduce:
1. WinXP Display Properties > Appearance > Font Size is "Normal"
2. On WinXP FF3 rc1, select Tools > Options... > Content


Actual Results:
Showed only small white space where current FF font size should display.

Expected Results:
Should show "Size: [ 14 ]" or whatever current font size is.

The same issue with the font size option occurs under Tools > Options > Content > Fonts & Colors > Advanced...

The current font size is indeed highlighted when the Size: list selection is clicked. So this is just a cosmetic slip. However, FF 2 show the current font size appropriately.
