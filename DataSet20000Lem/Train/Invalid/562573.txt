Pressing D in the File menu (Alt+F, D) close Firefox
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9.2.3) Gecko/20100401 Firefox/3.6.3
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9.2.3) Gecko/20100401 Firefox/3.6.3

In the File menu there is no shortcut defined for the D key, yet, when D is pressed while the File menu is open, Firefox close cleanly.

Reproducible: Always

Steps to Reproduce:
1. Open the File menu with the mouse or by pressing Alt+F.
2. Press the D key on the keyboard.
Actual Results:
The program closes, just a if "Exit" is selected from the File menu.

Expected Results:
Nothing.

Tried Firefox in safemode with all plugins disabled, yet this behaviour persists.
