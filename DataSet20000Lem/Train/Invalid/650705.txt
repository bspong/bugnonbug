"Bookmark all pages" menu option only appears through keyboard access
User-Agent: Mozilla/5.0 (Windows NT 5.1; rv:2.0) Gecko/20100101 Firefox/4.0
Build Identifier: Mozilla/5.0 (Windows NT 5.1; rv:2.0) Gecko/20100101 Firefox/4.0

If you left-click the Bookmarks menu using a mouse or a touchpad the "Bookmark all pages" option doe not appear. If you access the option through the keyboard (alt+B) it shows.

Reproducible: Always

Steps to Reproduce:
1.Left-click the Bookmark menu (mouse or touchpad)
2.The option "Bookmark all pages" will not appear
3.Access through keyboard. There it is.
Actual Results:
The option doe not show with mouse ortouchpad access, but it doe with the keyboard.

Expected Results:
Show the "Bookmark all pages" option when accessing through mouse or touchpad.
