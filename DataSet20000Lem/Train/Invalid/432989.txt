Bookmarks Toolbar folder not removable
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-GB; rv:1.9b5) Gecko/2008032620 Firefox/3.0b5
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-GB; rv:1.9b5) Gecko/2008032620 Firefox/3.0b5

The Bookmarks Toolbar folder cannot be deleted in Firefox 3b5.

In Firefox 2 the folder could be deleted by setting a new folder a the Toolbar folder. Fx3 ha no option to do this.

Reproducible: Always

Steps to Reproduce:
1. Click on Bookmarks
2. Right-click on "Bookmarks Toolbar"
3. Observe that "Delete" is greyed-out
4. Click on Organise Bookmarks
5. Observe that there is no option to alter the toolbar folder


Expected Results:
Offered a way to set a new folder a the Bookmarks Toolbar folder allowing the original Bookmarks Toolbar folder to be deleted.
