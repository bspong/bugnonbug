Back in Firefox 1.5, if I entered an empty query into the Wikipedia search in the search toolbar on the top-right corner, for example, I would be taken to the Wikipedia home page (http://en.wikipedia.org/wiki/Main_Page), but now with Firefox 2, when I enter an empty query, I'm taken to the search result for that empty query (http://en.wikipedia.org/wiki/Special:Search)

Reproducible: Always

Steps to Reproduce:
1. Select Wikipedia from the the search toolbar from the top-right corner.
2. Ensure search query is empty (delete any text there if nescessary)
3. Press enter.
Actual Results:
Taken to http://en.wikipedia.org/wiki/Special:Search

Expected Results:
Expected to be taken to http://en.wikipedia.org/wiki/Main_Page
