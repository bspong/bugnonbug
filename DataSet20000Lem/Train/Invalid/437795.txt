HTML Validator on a.m.o is not compatible with Firefox 3
The version of HTML Validator on a.m.o (0.8.4.0) is not compatible with Firefox 3 despite a new version exists (0.8.5.1) which is compatible with Fx3, see http://users.skynet.be/mgueury/mozilla/download.html. Please update the version on a.m.o.
