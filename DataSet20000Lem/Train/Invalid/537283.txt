Using window 7 "bump" feature to move resize a window to half of the screen, the suggestion for the search bar appear in a different location of the screen. The location is always the same place no matter where the firefox window started. Bumping the window to the left side of the screen the suggestion appear where they should be. Screenshot ha been attached to the bug report.

Reproducible: Always

Steps to Reproduce:
1.Launched firefox
2.'Bumped' window to right side of screen
3.Typed in search bar
Actual Results:
Suggestions appeared in different area of the screen

Expected Results:
Suggestions should have appeared directly under search bar

Running window 7 on an HP Pavilion dv6338se. Successfully reproduced bug using different search provider (yahoo, wikipedia), also tried using ebay search provider, which doe not provide suggestions, only search history and wa still able to reproduce bug.
