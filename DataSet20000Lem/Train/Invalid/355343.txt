Bookmark toolbar ha extra vertical space with google browser sync
User-Agent: Mozilla/5.0 (Macintosh; U; PPC Mac OS X Mach-O; en-US; rv:1.8.1) Gecko/20060918 Firefox/2.0
Build Identifier: Mozilla/5.0 (Macintosh; U; PPC Mac OS X Mach-O; en-US; rv:1.8.1) Gecko/20060918 Firefox/2.0

When google browser sync is turned in it introduces extra vertical space in the bookmark toolbar. This space is of a different color, creating a line. The total height of the toolbar increase when this is turned on also. Turning it off and restarting firefox remove the problem.

Reproducible: Always

Steps to Reproduce:
1. Install google browser sync
2. Turn it on.
3. Restart Firefox

Actual Results:
New line is introduced cause by additional vertical space in the toolbar.

Expected Results:
No change in the height of the toolbar is expected.
