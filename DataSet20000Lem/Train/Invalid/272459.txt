enabling Full Screen mode with no open window result in blank window covering entire desktop
User-Agent: Mozilla/5.0 (Macintosh; U; PPC Mac OS X Mach-O; en-US; rv:1.7.5) Gecko/20041107 Firefox/1.0
Build Identifier: Mozilla/5.0 (Macintosh; U; PPC Mac OS X Mach-O; en-US; rv:1.7.5) Gecko/20041107 Firefox/1.0

After closing all open windows, going to the menu -> View -> Full Screen result
in a large white window covering the entire desktop (menu remains visible though).

In addition, going to menu -> Toolbars, show a submenu that only ha a divider
and Customize... option (no Navigation Toolbar or Bookmark Toolbar).

To remove the white screen, go to menu -> View -> Full Screen again.

Reproducible: Always
Steps to Reproduce:
1. Close all browser windows.
2. Go to the menu -> View -> Full Screen

Actual Results:
A blank window/screen with no control covered the entire desktop.

Expected Results:
With no window open, the Full Screen option should be greyed out or a new
window in Full Screen mode should open.
