If a form lack an action then the default action in Firefox 3 includes any GET parameter in the URL. Firefox 2 (and all other browser I've used) omit the parameter in the default POST action.

Reproducible: Always

Steps to Reproduce:
1.Go to a page with GET params containing a form without an explicit action.
2.POST that form.

Actual Results:
The GET params are included in the URL posted to.

Expected Results:
The GET params should not be included, to match FF2 (and all other browsers).
