Crash if user drag a link from the page onto the Search Window.
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.7.6) Gecko/20050318 Firefox/1.0.2
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.7.6) Gecko/20050318 Firefox/1.0.2

As opposed to Firefox on Linux, Win32 version crash when the user drag the
link onto the search box in the web search bar.

Reproducible: Always

Steps to Reproduce:
1. Go to some web page
2. Get link, then drag it onto the Search Box
3. Observe the result

Actual Results:
Mozilla Firefox exit starting Talkback Client

Expected Results:
Initiate search for the text or link

Bug 100% exists in FF 1.0, 1.0.1, 1.0.2
