Taskbar preview and tab list on Windows 7 should be congruent to panorama tab-group
User-Agent: Mozilla/5.0 (X11; U; Linux i686; en-US; rv:1.9.2.13) Gecko/20101206 Ubuntu/10.10 (maverick) Firefox/3.6.13
Build Identifier: Mozilla/5.0 (Windows NT 6.1; rv:2.0b9) Gecko/20100101 Firefox/4.0b9

When a user is using Panorama and ha multiple tab groups, the Windows 7 taskbar preview should match only the current active tab group instead of simply listing all the tab open across all tab groups.

Reproducible: Always

Steps to Reproduce:
1. Create multiple tab groups.
2. Open multiple tab in each of the tab group.
3. Navigate to one of the tab group.
4. Hover over Firefox taskbar icon
Actual Results:
previews/list for ALL the tab is shown.

Expected Results:
Only the tab in the current tab group should be shown.
