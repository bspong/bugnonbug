Firefox display a squashed version of a site's favicon in the address bar
User-Agent: Mozilla/5.0 (Macintosh; U; PPC Mac OS X Mach-O; en-US; rv:1.8.1) Gecko/20061010 Firefox/2.0
Build Identifier: Mozilla/5.0 (Macintosh; U; PPC Mac OS X Mach-O; en-US; rv:1.8.1) Gecko/20061010 Firefox/2.0

Whenever you visit a site which ha a favicon, the icon is displayed squashed vertically in the address bar. However, favicons are displayed normally in the bookmark menu.

Reproducible: Always

Steps to Reproduce:
Visit a site with a favicon (eg. www.google.com).
Actual Results:
The favicon displayed in the address bar is squashed vertically.

Expected Results:
The favicon displayed in the address bar should be displayed normally at 16x16 and scaled down if necessary but not squashed.
