Bookmarks added or deleted in Firefox do not update the Favorites on Start Menu
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9.0.4) Gecko/2008102920 Firefox/3.0.4
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9.0.4) Gecko/2008102920 Firefox/3.0.4

If you activate to display the Favorites list on the Start Menu, addition and deletion made while using Firefox are not added or deleted from the Start Menu Favorites list.

Reproducible: Always

Steps to Reproduce:
1.Delete a bookmark in Firefox that wa imported from Explorer
2.Open the Favorites list from the start menu
3.The bookmark deleted in Firefox is still there.


Expected Results:
Deletions and addition to bookmark in Firefox should be relected in the Start Menu list of Favorites

The list of Firefox bookmark should MATCH the Start Menu list of Fasvorites
