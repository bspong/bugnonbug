Zoom from last tab persist when closed (same domain)
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9b2) Gecko/2007121120 Firefox/3.0b2
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9b2) Gecko/2007121120 Firefox/3.0b2

If you use CTRL+ in any tab and close it, the next tab will be show with the last zoom configuration

Reproducible: Always

Steps to Reproduce:
1. Open any URL
2. Create a new tab (CTRL+T)
3. Open another URL
4. Use CTRL+ any number of time
5. Close the tab
Actual Results:
The first URL will be show with the zoom used on the second URL

Expected Results:
The first URL must preserve the original zoom status
