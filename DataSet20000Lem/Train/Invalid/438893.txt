onclick event is fired for parent divs when clicking on child div in nested div situation

If you have a parent div that watch for onclick event and a child div that also watches, when you click on the child div, the onclick is fired and then when that code is finished the code for the onclick on the parent div is fired a well. For example try the following code.

<div onclick="alert('div1');" style="width: 100px;height: 100px;background-color: red;">
<div onclick="alert('div2');" style="width: 50px;height: 50px;background-color: blue;" />
</div>



Reproducible: Always

Steps to Reproduce:
1.Open page with provided html code.
2.Click on blue box
3.
Actual Results:
An alert will show that say "div2" followed by an alert that say "div1"

Expected Results:
Previous version would only have shown an alert saying "div2"
