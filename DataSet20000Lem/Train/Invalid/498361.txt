Firefox 3.0.11 remove previously saved bookmark
User-Agent: Mozilla/5.0 (Macintosh; U; Intel Mac OS X 10.5; en-US; rv:1.9.0.11) Gecko/2009060214 Firefox/3.0.11
Build Identifier: Mozilla/5.0 (Macintosh; U; Intel Mac OS X 10.5; en-US; rv:1.9.0.11) Gecko/2009060214 Firefox/3.0.11

Office environment with 50+ users. Some user previously running 3.0.8 others running 3.0.10. Some user running OS X 10.4.11 others running 10.5.7. All user had existing bookmark from previous version of Firefox. Affected user updated to or downloaded and replaced old version with 3.0.11. Under 3.0.11, all bookmark appear to be missing. Under 10.4, reinstalling older version restores previous bookmarks. Under 10.5, bookmark remain missing even when reverting to older version.

Reproducible: Always

Steps to Reproduce:
1. Have Firefox 3.0.8 - 3.0.10 and make bookmarks.
2. Update to or delete old and install 3.0.11
3. Look at bookmarks. They're gone.
Actual Results:
Had to revert to version prior to 3.0.11 on 10.4.11 system in order to get bookmark back. Bookmarks still missing under 10.5.7 system even with older version.

Expected Results:
Expected to update and standard thing like bookmark still be there.

No error logged to suggest why bookmark disappear.
