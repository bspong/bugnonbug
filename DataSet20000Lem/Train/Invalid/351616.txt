Toolbar in Bookmarks Manager styling is inconsistent with other toolbars
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8.1b2) Gecko/20060906 BonEcho/2.0b2
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8.1b2) Gecko/20060906 BonEcho/2.0b2

In the new 2.0 theme, toolbar button in Main Firefox window and Help window are styled without a bevel effect. In Bookmarks Manager, however, there is a bevel effect when the toolbar button are hovered.

Reproducible: Always

Steps to Reproduce:
1. Open Bookmarks Manager through Menu > Bookmarks > Organize Bookmarks..
2. Hover over the toolbar button

Actual Results:
There is a bevel effect when the toolbar button are hovered

Expected Results:
The toolbar button styling should be consistent with the Main Firefox window and Help window (currently no bevel effect)
