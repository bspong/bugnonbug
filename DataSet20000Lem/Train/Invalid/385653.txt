Setting current page a startpage make us the first open tab a current page
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; nl; rv:1.8.1.4) Gecko/20070515 Firefox/2.0.0.4
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; nl; rv:1.8.1.4) Gecko/20070515 Firefox/2.0.0.4

I tried to set another page a my startpage while I had three tab open. I wanted to use the page that wa on the second tab. The second tab wa active. I went to preference and pressed the button 'Use current page.' I expected that the page in the second tab would be used a my startpage, but the page in the first tab wa used.

Reproducible: Always

Steps to Reproduce:
1. Open two or more different tab
2. Select the second, the third, the fourth or the fifth tab
3. Go to Tools > Preferences
4. Press the button 'Use current page'
Actual Results:
The page that is loaded in the first tab is used a startpage.

Expected Results:
The page in the active tab should be set a startpage.
