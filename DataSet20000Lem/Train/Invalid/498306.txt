There is no spinning wait circle animation when "Always show tab bar" option is unchecked.

As far a I know, all web browser have always displayed an animation while waiting, so this should qualify a a bug. Previous firefox version did not have this problem.


Reproducible: Always

Actual Results:
There is no spinning wait circle animation when "Always show tab bar" option is unchecked.

Expected Results:
A wait animation should always be shown, regardless of whether the "Always show tab bar" option is checked.
