The vertical Scroll bar disappears on Windows XP Theme Change
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8.0.1) Gecko/20060111 Firefox/1.5.0.1
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8.0.1) Gecko/20060111 Firefox/1.5.0.1

When the Win XP theme is changed from default ot Classic, the vertical scroll bar within the browser (all tabs) disappears.

Reproducible: Always

Steps to Reproduce:
1. Set the Win XP theme to the default theme
2. Open firefox
3. Open couple of site in different tab
4. Minimize Firefox
5. Change the Win XP theme to "classic" theme
6. Maximize Firefox


Actual Results:
The window maximized with all the tabs, but NO vertical scroll bar.

Expected Results:
The vertical scroll bar should have been there.

- Installed extension :
1. Tabbrower pref.
2. Session Saver
