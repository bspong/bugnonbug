Having multiple language to option list disallows selecting the last one
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8.0.7) Gecko/20060909 Firefox/1.5.0.7
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8.0.7) Gecko/20060909 Firefox/1.5.0.7

In the "Languages in order of preference" choice list, having more than a few language will cause the list to not allow you to select the last item in the list with the mouse. This will work in the rare instance, but the majority of the time it will not allow selection of the last option.

Reproducible: Always

Steps to Reproduce:
1. Go to tab Tools -> Options -> Advanced -> General
2. Select "Edit Languages"
3. Add at least 7 language to the list
4. Try to click on the last one. It cannot be selected.


Actual Results:
Last language in list cannot be selected with the mouse.

Expected Results:
Selected the last item in the list.
