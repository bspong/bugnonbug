Regression: Reload icon display wrong when using small icon
User-Agent: Mozilla/5.0 (Macintosh; U; PPC Mac OS X Mach-O; en-US; rv:1.8) Gecko/20051111 Firefox/1.5
Build Identifier: Mozilla/5.0 (Macintosh; U; PPC Mac OS X Mach-O; en-US; rv:1.8) Gecko/20051111 Firefox/1.5

The Reload icon in the Navigation Toolbar is displayed wrong when you use
activate "Use Small Icons" and open a new tab, or open a link a a new window.

Reproducible: Always

Steps to Reproduce:
1. Customize Navigation Toolbar.
2. Activate "Show Small Icons".
3. Open a new tab.
Actual Results:
The blue Reload icon disappear and a some other graphic take it place.

Expected Results:
The blue Reload icon should be grayed out, just like when you use the "Normal"
icon size.
