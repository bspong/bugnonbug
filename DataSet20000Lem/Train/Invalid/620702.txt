Firefox menu disappears when loading Java applet in more than two tab
User-Agent: Mozilla/5.0 (Macintosh; Intel Mac OS X 10.6; rv:2.0b9pre) Gecko/20101221 Firefox/4.0b9pre
Build Identifier: Mozilla/5.0 (Macintosh; Intel Mac OS X 10.6; rv:2.0b9pre) Gecko/20101221 Firefox/4.0b9pre

If loading Java applet in more than two tabs, the Firefox menu bar is replaced with Java Applet Host menu bar. I am unable to access the FF menu or any FF button (List all tabs, Display all Bookmarks).

Reproducible: Always

Steps to Reproduce:
1.Open a page containing a Java applet (http://hg.mozilla.org/qa/litmus-data/raw-file/tip/firefox/popups/javaclock/index.html) in two different tabs.
2.Try to close one of the tabs.

Actual Results:
Firefox menu bar is replaced with Java Applet Host menu bar. I am unable to access the FF menu or to close any of the tabs.

Expected Results:
FF menu and button should be accessible.

I am using Java Plug-in 2 for NPAPI Browsers 13.3.0.
