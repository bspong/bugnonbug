Bookmarks icon (with the arrow) disappears from navigation bar when the menu bar is enabled
User-Agent: Mozilla/5.0 (X11; Linux x86_64; rv:2.0.1) Gecko/20100101 Firefox/4.0.1
Build Identifier: Mozilla/5.0 (X11; Linux x86_64; rv:2.0.1) Gecko/20100101 Firefox/4.0.1

The bookmark button, the one with the arrow, is disappeared from the navigation bar if it is placed there and the menu bar is enabled.

Reproducible: Always

Steps to Reproduce:
1.Customize toolbars by placing the bookmark icon (with the arrow) at navigation bar
2.Enable the menu bar from toolbars menu
3.The bookmark icon (with the arrow) disappears from navigation bar

Actual Results:
The bookmark icon (with the arrow) disappears from navigation bar

Expected Results:
The bookmark icon (with the arrow) stay visible on navigation bar

This icon if it is placed at another bar like menu bar or tab bar stay visible either the menu bar is enabled or not.
