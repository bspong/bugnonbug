Duplication of QUIP’s is occurring when same QUIP’s are entered more then once

Reproducible: Always

Steps to Reproduce:
1.Log in RecZilla application using valid admin account
2.Navigate to Candidate list page.
3.Click the “QUIP” displayed in the above portion of the page
4.Quip page will open, enter a new Quip and click Add this quip.
5.Follow the same procedure and add same Quip.


Actual Results:
Duplication of QUIP’s is seen.



Expected Results:
No same QUIP’s should be formed. And a message should be displayed “This Quip
already exists”
