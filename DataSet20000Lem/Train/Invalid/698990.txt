A blank page is displayed when performing a search with 1017 char or more using the Twitter search engine
Mozilla/5.0 (X11; Linux i686; rv:8.0) Gecko/20100101 Firefox/8.0
Build ID: 20111031184046

STR:
1. Launch Firefox.
2. Select the Twitter search engine in the search bar.
3. Enter a string 1017 character long or longer.
4. Click on the magnifying glass or press enter.

Expected Results:
An error message about the string being too long is displayed (as for Google) or the result list (as for the rest of the search engines).

Actual Results:
A blank page is displayed.

Note:
This issue reproduces on all OSs on Firefox 8.0, 9.0 and 10.0.
