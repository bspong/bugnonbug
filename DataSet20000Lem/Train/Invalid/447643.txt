Printing from print preview button cause back button to become unavailable
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9.0.1) Gecko/2008070208 Firefox/3.0.1
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9.0.1) Gecko/2008070208 Firefox/3.0.1

Anytime a page is printed from the print preview menu, the back navigation button cannot be selected.

Reproducible: Always

Steps to Reproduce:
1.Enter a valid URL into the address bar, and hit enter.
2.Enter another valid URL into the address bar, and hit enter.
3.Select File -> Print Preview
4.Print the page from within the print preview page.

Actual Results:
When returned to Firefox, the back navigation button will be unavailable (grey), clicking it ha no effect. However the backspace key will navigate to the previous page.

Expected Results:
The back navigation button should be available.
