When using the Chromifox Basic theme, the Details arrow button of the Clear Recent History dialog box is blank and is the default rectangle shape.

Reproducible: Always

Steps to Reproduce:
1. Install and use the Chromifox Basic theme.
2. Open the Clear Recent History dialog box.
Actual Results:
Details arrow button is blank and is the default rectangle shape.

Expected Results:
The button should be square and display the picture of an arrow.
