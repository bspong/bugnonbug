JavaScript Console should be cleared when clicking "clear all" from Options --> Privacy
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.7.6) Gecko/20050317 Firefox/1.0.2
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.7.6) Gecko/20050317 Firefox/1.0.2

Because the JavaScript console can be used to see which site have been visited,
it would be nice if it could be cleared from the Options --> Privacy screen.

Reproducible: Always

Steps to Reproduce:
1. Select "Options..." from the "Tools" menu.
2. Select "Privacy".
3. Click the "Clear All" button and then click the "OK" button.
4. Select "JavaScript Console" from the "Tools" menu.
Actual Results:
Any site you've visited that contained JavaScript error are shown.

Expected Results:
Clear the JavaScript console.
