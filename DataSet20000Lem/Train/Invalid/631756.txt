Embedded Window Media Player plugin Litmus test: I don't have the Windows Media Player plugin installed and it seems there's no Windows 7 version, so I can't install it.

Reproducible: Always

Steps to Reproduce:
1. Browse to https://litmus.mozilla.org/run_tests.cgi?test_run_id=260
2. Select Windows 7, es-AR
3. Click on "Submit Configuration".
4. Select "4.0 Web App Compatibility" and "FX 4.0 - Common Operations".
5. Click on "Submit".
6. Select "3: Embedded Window Media Player plugin test".
7. Confirm you don't have the plugin installed.
8. Middle click the link "this instructions".
9. Scroll down to "Installing the plugin" and "1. Go to the plugin download page on Port25".
10. Middle click the link "plugin download page on Port25".

Actual Results:
There's no Windows 7 version.

Expected Results:
There should be a Windows 7 version.
