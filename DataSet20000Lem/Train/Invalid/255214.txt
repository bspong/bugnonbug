Cannot select print stylesheets in print preview
User-Agent: Mozilla/4.0 (compatible; MSIE 6.0; Windows NT 5.1) Opera 7.53 [en]
Build Identifier: Firefox 0.9.3

Print Preview on mozilla.org : the print preview
is shown supposedly using the print.css, but
the stylesheet selection icon in the statusbar only show
the stylesheets available for the "screen" and "all" medias,
whereas it should show those for "print" and "all".

Reproducible: Always
Steps to Reproduce:
1. Open www.mozilla.org with firefox
2. File/Print Preview
3. Right click on the stylesheet selector in the status bar

Actual Results:
It display the stylesheets for "screen" and "all" medium

Expected Results:
Displaying those for "print" and "all"
