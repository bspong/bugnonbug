"Clear All" command in Tools>Options>Privacy crash Firefox
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.7.8) Gecko/20050511 Firefox/1.0.4
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.7.8) Gecko/20050511 Firefox/1.0.4

The "Clear All" command cause Firefox to crash.

Reproducible: Always

Steps to Reproduce:
1.Browse to build up cache file
2.Select "Clear All" from Tools>Options>Privacy.


Actual Results:
Windows report Firefox unresponsive and it closes.

Expected Results:
History, cache, cookies, etc. cleared.

Workaround: Clear cache first manually, then use Clear All.
