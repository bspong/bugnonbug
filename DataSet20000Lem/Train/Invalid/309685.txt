Deerpark Beta 1 cannot close first tab
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8b4) Gecko/20050908 Firefox/1.4
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8b4) Gecko/20050908 Firefox/1.4

In Firefox 1.5b1, it is impossible to close the first tab when multiple tab are
open.

Reproducible: Always

Steps to Reproduce:
1. Open Firefox 1.5b1
2. Open another tab
3. Attempt to close the first tab using any method

Actual Results:
The tab did not close

Expected Results:
The tab should have closed
