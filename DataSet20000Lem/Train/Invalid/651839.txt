"When I open a link in a new tab, ..." checkbox work in opposite
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9.2.16) Gecko/20110319 Firefox/3.6.16 ( .NET CLR 3.5.30729; .NET4.0E)
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9.2.16) Gecko/20110319 Firefox/3.6.16 ( .NET CLR 3.5.30729; .NET4.0E)

Affect just hotkeys

Reproducible: Always

Steps to Reproduce:
1. Open firefox browser.
2. Open any page.
3. Go to tools->tools->options->Tabs menu item.
4. Check "When I open a link in a new tab, switch to it immediately" checkbox.
5. Close "Options" window.
6. Click on any link with pressed "Shift + ctrl" keyboard combination.



Actual Results:
New link open in new tab, but browser doesn't switch to it.

Expected Results:
New link open in new tab, and browser switch to it.

Opposite behavior is if uncheck "When I open a link in a new tab, switch to it immediately" checkbox.
