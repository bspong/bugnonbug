User-Agent: Mozilla/5.0 (Windows NT 5.1; rv:2.0b9) Gecko/20100101 Firefox/4.0b9
Build Identifier: Mozilla/5.0 (Windows NT 5.1; rv:2.0b9) Gecko/20100101 Firefox/4.0b9

Since upgrading to Firefox 4 beta 9 after starting Firefox arrow on back and forward button are hidden when using small icons, button (borders) are visible. Opening any website cause back arrow to appear, after clicking back button, arrow on forward button appears too. If I open add-on menager or Customize Toolbar window both arrow appears. Arrows are displayed till restarting firefox. Problem didn't occurs when using big (normal size) icon (so in safe mode also) or Firefox ha been started with more then one tab opened.

Reproducible: Always

Steps to Reproduce:
1. Open Firefox
Actual Results:
Arrows on forward and back button aren't visible

Expected Results:
Arrows on forward and back button are visible

Problem occurs with all add-on disabled and with default theme.
