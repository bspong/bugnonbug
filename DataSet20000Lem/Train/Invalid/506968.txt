Cannot edit personal information in Facebook
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9.1.1) Gecko/20090715 Firefox/3.5.1
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9.1.1) Gecko/20090715 Firefox/3.5.1

When attempting to edit personal information in your Facebook profile, the user can not drag down the window (in edit mode) to allow the user to edit text below the visible window.

Reproducible: Always

Steps to Reproduce:
1. Navigate to Facebook account.
2. Select Profile option, then select Info tab.
3. Select the option to Edit personal information.
4. If the user ha for any entry (such a favorite movies, favorite books, etc.) text that is below the window, the user can neither scroll down or drag window to larger dimension.
Actual Results:
1. Navigate to Facebook account.
2. Select Profile option, then select Info tab.
3. Select the option to Edit personal information.
4. If the user ha for any entry (such a favorite movies, favorite books, etc.) text that is below the window, the user can neither scroll down or drag window to larger dimension.

Expected Results:
1. Navigate to Facebook account.
2. Select Profile option, then select Info tab.
3. Select the option to Edit personal information.
4. If the user ha for any entry (such a favorite movies, favorite books, etc.) text that is below the window, the user should be able to navigate down to the text to edit.

1. Windows XP SP3...Firefox 3.5.1
2. Extensions - ABP, ABP with Element Hiding, BetterPrivacy, Gmail Manager
3. This behavior is replicated with ADP disabled.
4. This behavior is replicated with Firefox invoked in Safe Mode.
5. This behavior is replicated after the most aggressive Firefox uninstall option in Revo Uninstaller, then reinstall Firefox without installing add-ons.
6. No image exception are set in content loading.
7. This feature work fine in Google Chrome beta 3.0.193.2
