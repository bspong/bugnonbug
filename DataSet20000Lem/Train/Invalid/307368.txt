Bookmark unselectable by keyboard navigation in sidebar when only one bookmark listed
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8b4) Gecko/20050907 Firefox/1.4
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8b4) Gecko/20050907 Firefox/1.4

Bookmark unselectable by keyboard navigation in sidebar when only one bookmark
listed.

Reproducible: Always

Steps to Reproduce:
1. Open up bookmark sidebar (CTRL-B)
2. Search bookmark until only one bookmark is listed
3. Press tab key (to focus the bookmark listbox)
4. Press the down arrow key to select the one bookmark
Actual Results:
Can not select the bookmark

Expected Results:
Bookmark is selected
