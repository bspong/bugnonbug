Back/Forward button glitch in the bookmark window with OSX default theme

Build Identifier: Mozilla/5.0 (Macintosh; U; Intel Mac OS X 10.5; en-US; rv:1.9.0.8) Gecko/2009032608 Firefox/3.0.8

The "Go Back/Go Forward" toolbar button in the "Bookmarks -> Organize Bookmarks" window have some graphic glitch and change position depending on whether they are active or not. This with the default theme on Mac OS X.

Reproducible: Always

Steps to Reproduce:
1. Open the "Bookmarks -> Organize Bookmarks" window and note the "Go Forward" button isn't completely drawn;
2. Click on a folder to make the "Go Back" button become active;
3. No "Go Back" and note how the "Go Forward" button change position and now partially overlap the "Go Back" button.
