firefox fails to indicate that it is loading a website
User-Agent: Mozilla/4.0 (compatible; MSIE 6.0; Windows NT 5.1; SV1; .NET CLR 1.1.4322)
Build Identifier: Mozilla/5.0

When visiting any website, in previous build the connection status is
displayed down in the status bar on the browser, and the icon in the top-right
of the browser animates. Neither of these operation perform when a website is
loading.

Reproducible: Always
Steps to Reproduce:
1. Visit website
2. No animation of icon, no indication in status bar

Actual Results:
No animation of icon, no indication in status bar

Expected Results:
Icon should animate, along with the status of visiting the website in the
status bar
