User Agent: Mozilla/5.0 (X11; Linux i686; rv:7.0.1) Gecko/20100101 Firefox/7.0.1
Build ID: 20110928134238

Steps to reproduce:

Enter something into search box. Click somewhere else so that the search box loos the focus. Go back to the search box by doing a right-click on it.


Actual results:

Typing "I" cursor appears on the search box.


Expected results:

The text in the search box should get selected automatically, just like in Windows or Mac version of Firefox.
