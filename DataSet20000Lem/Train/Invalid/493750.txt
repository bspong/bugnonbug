Serbian TLD is wrong in Preferred Language Dialog
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 6.0; en-GB; rv:1.9.0.10) Gecko/2009042316 Firefox/3.0.10 (.NET CLR 3.5.30729)
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 6.0; en-GB; rv:1.9.0.10) Gecko/2009042316 Firefox/3.0.10 (.NET CLR 3.5.30729)

Tools > Options > Content > Languages. In the "Select a language to add" Serbian country code (TLD) show 'sr', but the Serbian TLD is actually .rs. .sr stand for Suriname. See http://en.wikipedia.org/wiki/Cctld

Reproducible: Always

Steps to Reproduce:
1. Go to Tools > Options > Content > Languages
2. Open Select language to add.
Actual Results:
In the "Select a language to add" Serbian country code (TLD) show 'sr'.

Expected Results:
The Serbian TLD is actually .rs, so it should read "Serbian [rs]".
