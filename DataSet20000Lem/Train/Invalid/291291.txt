Text in form doesn't show text select cursor when onchange alert is shown
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.7.7) Gecko/20050414 Firefox/1.0.3
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.7.7) Gecko/20050414 Firefox/1.0.3

If you have a textbox which show an alert onchange, the text in that form then
doe not show the Text Select Cursor to indicate that you are hovering over text.

Reproducible: Always

Steps to Reproduce:
1.Trigger on Change Event
2.Click OK on Alert Box
3.Hover over text in same form

Actual Results:
Text Select Cursor not shown when hovering over text, normal cursor shown instead.

Expected Results:
To see Text select cursor when hovering over text.
