When using Walnut theme, the "iconless" item in browser menu and context menu
sometimes are aligned to the left, thus starting in the "icon gutter" area

Reproducible: Always
Steps to Reproduce:
1. install Walnut theme
2. rerun Firefox
3. open any page
4. right-click anywhere on a page
Actual Results:
Iconless item start directly in the icon strip

Expected Results:
Start them a regular item

It doesn't happen in the default and Cats theme
