Closing a pop up window force multiple tab close in Firefox 4
User-Agent: Mozilla/5.0 (Windows NT 6.1; WOW64; rv:2.0) Gecko/20100101 Firefox/4.0
Build Identifier: Mozilla/5.0 (Windows NT 6.1; WOW64; rv:2.0) Gecko/20100101 Firefox/4.0

On certain website where pop up window occur, when trying to close the pop up window, Firefox 4 pop up with the message, "You are about to close X tabs, do you want to continue? Ok or Cancel?"

This wa not a problem with Firefox 3 but since installing Firefox 4 it happens all the time.

Reproducible: Always

Steps to Reproduce:
1. Open a website with a pop up window
2. Close the pop up window
3. Press cancel in the message window to continue browsing
Actual Results:
A pop up message window pop up asking to close multiple tab

Expected Results:
No pop up message window should have occurred asking to close multiple tabs.
