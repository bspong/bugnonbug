IDN displayed in ACE format in location bar
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; pt-BR; rv:1.8.1.3) Gecko/20070309 Firefox/2.0.0.4
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; pt-BR; rv:1.8.1.3) Gecko/20070309 Firefox/2.0.0.4

When you open an international domain name in Firefox, a soon a you hit "Enter", what you typed turn into the ACE format instead of keeping the typed address. So, if you type: http://www.tchêbian.com, Firefox will display: http://www.xn--tchbian-lya.com, which is rather unaesthetic.

Reproducible: Always

Steps to Reproduce:
1. Type http://www.tchêbian.com in the location bar
2. Hit "Enter"
Actual Results:
Firefox display http://www.xn--tchbian-lya.com in the location bar.

Expected Results:
Keep http://www.tchêbian.com a is in the location bar.

More info on the ACE format: http://sedo.com/idn/
