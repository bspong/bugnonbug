Firefox3 ignores the "show a blank page on startup " option
User-Agent: Mozilla/5.0 (X11; U; Linux i686; en-GB; rv:1.9) Gecko/2008051202 Firefox/3.0
Build Identifier: Mozilla/5.0 (X11; U; Linux i686; en-GB; rv:1.9) Gecko/2008051202 Firefox/3.0

Firefox3 seems to ignore the "Show a blank page on startup" option.

Reproducible: Always

Steps to Reproduce:
1. Edit->Prefernces->Main
2. Set "When firefox starts" to "Show a blank page"
3. Open some random websites, each in a separate tab
4. Restart Firefox

Actual Results:
Firefox load all the tab that were opened the last time.

Expected Results:
Firefox should display an empty page.
