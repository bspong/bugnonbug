Vista: New Default Programs API's and setting better integrate Firefox into the Windows Shell
Vista support a new set of API’s and registry setting for registering default applications, “related applications”, mime types, and setting related to the detailed product description available to the user through the Default Programs applet on the Vista start menu. Using these method to register Firefox will better integrate Firefox into the shell.
