IE not found by german version
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; de; rv:1.8) Gecko/20051111 Firefox/1.5
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; de; rv:1.8) Gecko/20051111 Firefox/1.5

IE won´t be launched in the german version of Firefox 1.5 because it look for it
under /Program Files/Internet Explorer. A typical german installation of IE will store it under /Programme/Internet Explorer.
Workaround: Copy folder Internet Explorer to Program Files.

Reproducible: Always
