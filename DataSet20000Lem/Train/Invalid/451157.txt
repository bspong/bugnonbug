Text below icon in Preferences window is invisible when the system theme is set to HighContrastInverse theme.
This bug can be found with firefox 3.0.1 and trunk build.

Steps to reproduce:
1. Set system theme to HighContrastLargePrintInverse theme.
2. Start firefox, go to menu Edit->Preferencese.
3. Hover mouse cursor above each preference tab.

Bug observations:
The text below each tab icon, say "Main", "Tabs" and etc., is invisible when mouse cursor is moved above.
