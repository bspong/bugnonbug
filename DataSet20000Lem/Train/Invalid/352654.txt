When a separator is deleted, the "Edit" menu in the Bookmark Manager show "Undo Delete Bookmark" instead of "Undo Delete Separator"

Reproducible: Always

Steps to Reproduce:
1. Create a new separator
2. See the option under Edit menu - "Undo New Separator" would be shown
3. Now delete a Separator. Instead of "Undo Delete Separator" it show "Undo Delete Bookmark"

Actual Results:
"Undo Delete Separator" should appear under "Edit" menu

Expected Results:
"Undo Delete Bookmark" appearing
