Buttons for navigation toolbar appear 4 time under each other in Mozilla Firefox 4 beta 1
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 6.1; WOW64; en-US; rv:2.0b1) Gecko/20100630 Firefox/4.0b1
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 6.1; WOW64; en-US; rv:2.0b1) Gecko/20100630 Firefox/4.0b1

When starting Mozilla Firefox 4 beta 1 it can be noticed that the button in the navigation toolbar appear 4 time under each other.

It can also be noticed that the button don't work. The navigation toolbar is one hotspot, clicking it cause the bookmark menu to appear.

Reproducible: Always

Steps to Reproduce:
1. Launch Mozilla Firefox 4 beta 1
2. View the navigation toolbar (it ha 4 set of same buttons)
3. Move the mouse over the navigation toolbar
4. Observe that the complete toolbar is highlighted
5. Observe that when clicking the navigation toolbar, the bookmark menu is opened
Actual Results:
Complete navigation toolbar highlight when moving the mouse pointer over the toolbar.

Clicking the toolbar cause that the bookmark menu is displayed.

Expected Results:
That the button in the navigation toolbar are only displayed once.
That the button work.
That the bookmark menu is not displayed when clicking the toolbar.
