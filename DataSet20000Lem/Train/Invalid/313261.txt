Installing OSX Firefox stop Safari RSS working
User-Agent: Mozilla/5.0 (Macintosh; U; PPC Mac OS X Mach-O; en-US; rv:1.7.12) Gecko/20050915 Firefox/1.0.7
Build Identifier: Mozilla/5.0 (Macintosh; U; PPC Mac OS X Mach-O; en-US; rv:1.7.12) Gecko/20050915 Firefox/1.0.7

If Firefox (1.0.7 or 1.5b2)is installed without being set a default, it stop
Safari 2.0 RSS article link from being read. When an article is clicked on in
Safari, it open Firefox, which cannot read the format. The only way to restore
the Safari functionality is to remove all instance of Firefox from the Mac.

Reproducible: Always

Steps to Reproduce:
1.Install Tiger (to get Safari 2.0)
2.Install stable or beta Firefox
3.Click on an article link in Safari RSS
4. Firefox open even when Safari is default browser.


Actual Results:
Get error that Firefox cannot read the format of the feed.

Expected Results:
Nothing, Firefox should not have anything to do with Safari RSS articles!
