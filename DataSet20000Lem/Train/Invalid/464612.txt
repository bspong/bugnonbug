Firefox 3.0.4 installer launch program without profile
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 6.0; en-US; rv:1.9.0.4) Gecko/2008102920 Firefox/3.0.4
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 6.0; en-US; rv:1.9.0.4) Gecko/2008102920 Firefox/3.0.4

Launching Firefox a final install step cause program to run without existing profile

Reproducible: Always

Steps to Reproduce:
1. login a standard user
2. launch Firefox Setup 3.0.4 a administrator to upgrade existing program (3.0.3)
3. complete setup after giving UAC password
4. select run Firefox checkbox on final installer screen
5. Firefox start without existing profile information.
Actual Results:
Firefox start without existing profile information.

Expected Results:
Firefox run with existing user profile.

If one close FF and restarts it from the standard user account profile is restored.
