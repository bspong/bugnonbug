No cut/copy/paste/delete option in context menu in text input field
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9) Gecko/2008052906 Firefox/3.0
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9) Gecko/2008052906 Firefox/3.0

The cut, copy, paste and delete option aren't present when right clicking inside text input field to display the context menu. This first began with version 3.0; v2.0.0.14 had these option in the context menu. I don't know if these item were puposely removed in FF3 or if this is a bug.

Reproducible: Always

Steps to Reproduce:
1. Locate a webpage with a text or textarea form input field.
2. With the cursor of the text field, click the right mouse button to display the context menu.
Actual Results:
No cut, copy, paste or delete item listed in the context menu.

Expected Results:
Cut, copy, paste and delete should display in text field's context menu.
