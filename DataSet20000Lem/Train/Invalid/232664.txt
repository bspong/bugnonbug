When afer clicking link web page is not opened and dialaog appears (e.g. The
connection wa refused when attempting to conect...), after closing the dialog
it is not possible to open that link again with mouse click (both with left and
middle). But it is possible to open it via context many "open in new tab".

Tabbrowser extension must be installed!!!

Reproducible: Always
Steps to Reproduce:
0. Install tabbrowser extension plugin
1. click on any link to open page
2. brake the connection before page download is complete
3. click OK on the message box
4. make the connection again
5. click on the link again

Actual Results:
After clicking again there won't be any reaction.

Expected Results:
The page should be opened normaly
