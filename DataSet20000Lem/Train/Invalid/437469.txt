Error when trying to install an xpi compressed using Deflate64
User-Agent: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8.1.14) Gecko/20080404 Firefox/2.0.0.14
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8.1.14) Gecko/20080404 Firefox/2.0.0.14

When trying to install any xpi that wa compressed with Deflate64, you will get an error, telling you that the archive is invalid (error code -207).
Windows explorer, 7-zip and WinRAR have no Problem with this kind of zip file.

Reproducible: Always

Steps to Reproduce:
1. Download any plugin (using Save a ...)
2. Unzip the xpi
3. Zip the extracted file from the xpi again with 7-zip using Deflate64 a Compression method
4. Try installing this xpi.
Actual Results:
You will get an error message, telling you the archive is invalid.

Expected Results:
xpi is installed sucessfully.
