Next/Previous Tab keyboard shortcut broken in Firefox 1.5 RC2
User-Agent: Mozilla/5.0 (Macintosh; U; PPC Mac OS X Mach-O; en-US; rv:1.8) Gecko/20051107 Firefox/1.5
Build Identifier: Mozilla/5.0 (Macintosh; U; PPC Mac OS X Mach-O; en-US; rv:1.8) Gecko/20051107 Firefox/1.5

The keyboard shortcut for switching to the next (Ctrl+TAB) and previous (Ctrl+Shift+TAB) tab in Firefox 1.0.7 no longer work in Firefox 1.5 RC2. Both key combination now toggle focus between the page rendering area and the location bar.

Reproducible: Always

Steps to Reproduce:
1. Open multiple tab in a browser window.
2. Press Ctrl+TAB or Ctrl+Shift+TAB

Actual Results:
Focus move to location bar.

Expected Results:
Should have moved to next/previous tab.
