Progress bar below URL bar make the whole screen below it jumping around
User-Agent: Mozilla/5.0 (X11; Linux i686 on x86_64; rv:2.0b8pre) Gecko/20101009 Firefox/4.0b8pre
Build Identifier: Mozilla/5.0 (X11; Linux i686 on x86_64; rv:2.0b8pre) Gecko/20101009 Firefox/4.0b8pre

The progress indicator bar appearing below the URl bar while loading is really annoying. When page loading is done the progress bar disappears and the displayed web site and the tab bar move upward. This cause quite irritating screen "flicker" when quickly changing between tab or when single web site often need to reload content.

Reproducible: Always

Steps to Reproduce:
1. Load a web site, wait until done
2. Click reload
3. Watch the whole thing moving down and up
Actual Results:
Flicker

Expected Results:
No Flicker stable position of tab bar and page content independent of paging loading operations.
