button type="submit" marked a required in an input is required
User-Agent: Mozilla/5.0 (Windows NT 6.1; rv:2.0b8pre) Gecko/20101011 Firefox/4.0b8pre
Build Identifier: Mozilla/5.0 (Windows NT 6.1; rv:2.0b8pre) Gecko/20101011 Firefox/4.0b8pre

If you are using a button in your form and have it set to type="submit" and you have an input in the form using the required attribute Firefox put a red border around it indicating it is required. This ha only started happening in 4.0b8pre.

Reproducible: Always

Steps to Reproduce:
1. Create a form
2. Put in an input and add the required attribute
3. Add a button with the type set to submit
4. View in Firefox 4.0b8pre
Actual Results:
Red border is displayed around the button

Expected Results:
There should be no red border around the button
