URL shown in the status bar in Bookmarks Manager is not immediately updated after modification of a URL of an item
Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9a8pre) Gecko/2007081208 Minefield/3.0a8pre ID:2007081208 [cairo]

1. Bookmarks | Organize Bookmarks...
2. Right-click an item -> Properties
3. Modify Location
4. Save Changes

The URL in the status bar is still the one before the modification. You have to move the cursor and back to the modified item to see the new URL in the status bar.
