"Smart Bookmarks" folder not created any more

With Firefox 3.0.3 (windows and linux build), the "Smart Bookmarks" folder is not created in the Bookmarks Toolbar any more. 

Reproducible: Always

Steps to Reproduce:
1. Run "firefox -P" and create a new profile.
2. Start firefox using the new profile.


Actual Results:  
"Smart Bookmarks" is not seen on the bookmarks toolbar.

Expected Results:  
There is a "Smart Bookmarks" folder on the bookmarks toolbar.
