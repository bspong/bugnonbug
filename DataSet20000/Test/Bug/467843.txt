If I open a new tab in firefox, the address bar still points to the old tab. So if I type a new web address, the webpage opens in old tab. I observed this bug only after updating to ver 3.0.4.

Reproducible: Always

Steps to Reproduce:
1.Open firefox (ver 3.0.4)
2.open a new tab without closing the older tabs 

Actual Results:  
the address bar still points to the old tab

Expected Results:  
the address bar should be pointing the newly opened tab (i.e. blank)

If  two new tabs are opened and if the 1st new tab is pointed, it becomes active (i.e the address bar becomes blank and webpage opens in the 1st new tab)
