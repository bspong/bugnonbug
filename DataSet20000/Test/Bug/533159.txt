Hyperlinks are not working if placed before a label
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 6.1; en-GB; rv:1.9.1.5) Gecko/20091102 Firefox/3.5.5
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 6.1; en-GB; rv:1.9.1.5) Gecko/20091102 Firefox/3.5.5

While using Visual studio 2008... (The components mentioned below, are the default visual studio 2008 components)

If i hyperlink component is placed before or in the middle of label components, then the hyperlink component is treated like a pure label in firefox.

Everything works like a charm if i place the hyperlink component after a label component. 

Reproducible: Always

Steps to Reproduce:
1. Place a hyperlink component from visual studio
2. Place a label component after the hyperlink component



In the example below the hyperlink is not treated right

<asp:HyperLink ID="hyperlink" runat="server" NavigateUrl="~/test.aspx">This should work as hyperlink</asp:HyperLink><asp:Label ID="lblShmeio3" runat="server" Text="This one is the label"></asp:Label>

While in this case it works ok.

<asp:Label ID="lblShmeio3" runat="server" Text="This one is a label"></asp:Label><asp:HyperLink ID="hyperlink" runat="server" NavigateUrl="~/test.aspx">This should work as hyperlink</asp:HyperLink>
