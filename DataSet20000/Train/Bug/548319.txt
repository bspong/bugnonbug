Denied access on T-Mobile hotspots (Germany)
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 6.1; de; rv:1.9.1.8) Gecko/20100202 Firefox/3.5.8
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 6.1; de; rv:1.9.1.8) Gecko/20100202 Firefox/3.5.8

Access on T-Mobile hotspots can not be established. After entering the access information following error message shows up (this is a translation from German): 
"Error: Network Timeout - server at mein.t-mobile.de needs too long to send an answer".
The access via Internet Explorer can be established without any problem on the same computer at the same time.
This error has been encountered in the last 8 weeks continuously every time when I tried to log onto the T-Mobile hotspot at Frankfurt airport and at Munich airport. 

Reproducible: Always

Steps to Reproduce:
1. Go to a German T-Mobile hotspot, preferably at airport lounges (in the cases encountered it was the Lufthansa First Class lounges
2. After establishing access with the hotspot go to www.t-mobile.de
3. Enter login data
Actual Results:  
Error message as shown in details
German: 
Fehler: Netzwerk-Zeitüberschreitung
Der Server unter mein.t-mobile.de braucht zu lange, um eine Antwort zu senden.

Expected Results:  
Regular login confirmation page
