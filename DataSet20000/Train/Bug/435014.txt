Español:
Cuando abro una nueva pestaña con una nueva página, o cuando ejecuto Firefox y restauro la última sesión, las pestañas no tienen la dirección en la barra de dirección. Entonces, si detengo la página, ésta queda en blanco y no hay forma de conocer la dirección.

English:
When I open a new tab with a new page, or when I run Firefox and restrore the last session, the tabs don't have the address in the address bar. So, if I stop the pages, it remains blank and there's no way to know the address.


Reproducible: Always

Steps to Reproduce:
Español:
1.Abro una nueva página en una nueva ventana (usando Ctrl+Click o cuando restauro la sesión);
2.Detengo la página luego de unos segundos;
3.Actualizo la página.

English:
1.Open a new page in a new tab (using Ctrl+Click or when I restore the session);
2.Stop the page after a few seconds;
3.Refresh the page.
Actual Results:  
Español: La página no se actualiza y continúa vacía y la barra de direcciones también.
English: The page don't refresh and remains blank  and also the address bar.


Expected Results:  
Español: La barra de direcciones debería tener la dirección y al actualizar mostrar la página.
English: The address bar must have the address and when I refresh, it must show the page.

Cuando sucede:
*Presiono Ctrl+Click para abrir una nueva página en una nueva pestaña: Falla
*Abro un enlace en la misma pestaña: Creo que falla
*Escribo la dirección: La dirección continúa, pero F5 no hace nada (enter sí)
*Restauro la sesión: Falla

When happends:
*I Ctrl+Click to open a new page in a new tab: Fails
*I open a link in the same tab: I think it fails
*I write the address: The address remains, but F5 don't refresh (enter loads).
*I restore the session: Fails
