Cannot create a new bookmark folder on clicking the New Folder button
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9.1.4) Gecko/20091016 Firefox/3.5.4 (.NET CLR 3.5.30729)
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9.1.4) Gecko/20091016 Firefox/3.5.4 (.NET CLR 3.5.30729)

When I bookmark a page and want to store it to a new folder in the Bookmarks Menu and I click the New Folder button, the Bookmarks Toolbar and Bookmarks Menu gets highlighted and the New Folder button is disabled but no new folder is created

Reproducible: Always

Steps to Reproduce:
1. Double click the bookmark icon on the address bar opens up the bookmark window
2. Expand the Bookmarks Menu to view folders. 
3. Click the New Folder button
4. The labels Bookmarks Toolbar and Bookmarks Menu become highlighted
5. The New Folder button becomes disabled
6. No new folder is created
Actual Results:  
Unable to create a new folder and save the bookmark to a new folder. Unable to rename a new folder by double clicking it either.

Expected Results:  
A new folder is created at the bottom of the window (under the Bookmarks Menu). I can rename the folder and click save to save my bookmark.

Have noticed this problem in the earlier version of Firefox as well. I have recently upgraded to 3.5.4
