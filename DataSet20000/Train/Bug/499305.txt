status bar(bottom,right) - drag option to increase or decrease the window size is not consistent.
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9.0.11) Gecko/2009060215 Firefox/3.0.11
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9.0.11) Gecko/2009060215 Firefox/3.0.11

1,In status bar(bottom, right)- drag option to increase or decrease the window size is not consistent. Some times browser not displaying that option.
2,While using that option displays "Restore Down" button instead of "maximize" button.

Reproducible: Sometimes

Steps to Reproduce:
1.Install firebug plug-in and restart the browser
2.Type any URL(www.google.com) in location bar and press enter.
3.Press F12 or click firebug icon to open firebug window.
4.Click Script menu in firebug window.
5.Change the option.(If Enabled click Disabled)
6.Now "drag option to increase or decrease the window size" is not available.
Actual Results:  
1.drag option to increase or decrease the window size is not consistent.
2.While using that option displays "Restore Down" button instead of "maximize" button

Expected Results:  
1.drag option to increase or decrease the window size should be available.
2.While using that option, should display "maximize"
