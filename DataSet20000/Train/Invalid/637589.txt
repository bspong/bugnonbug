Firefox process doesn't terminate in Windows 7 SP 1
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 6.1; de; rv:1.9.2.13) Gecko/20101203 Firefox/3.6.13 (.NET CLR 3.5.30729)
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 6.1; de; rv:1.9.2.13) Gecko/20101203 Firefox/3.6.13 (.NET CLR 3.5.30729)

After installing SP 1 for Windows 7 Firefox process doesn't terminate after closing the window. The window closes but the process hangs and stays in the tasklist. One must kill the process using task manager before Firefox can be started again.

Reproducible: Always

Steps to Reproduce:
1. Download and install SP 1 for Win 7
2. Open Firefox and close it
3. Firefox process stays in task list
