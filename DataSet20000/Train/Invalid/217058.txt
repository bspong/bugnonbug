After selecting Cache under Preferences>Advanced, no other selections respond
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.5a) Gecko/20030728 Mozilla Firebird/0.6.1
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.5a) Gecko/20030728 Mozilla Firebird/0.6.1

If user right clicks on the Firebird icon and selects
Preferences>Advanced>Cache, no other selection after selecting Cache changes any
of the options that are available under Preferences.

Reproducible: Always

Steps to Reproduce:
1.Right Click Firebird icon
2.Select Preferences>Advanced>Cache
3.Try to select any other options under any category

Actual Results:  
Same as described. The only way to choose any other Category after
Preferences>Advanced>Cache, is to close the window and start over.

Expected Results:  
After selecting Cache, the user should be able to proceed on changing any of the
other Preferences.

I was running Firebird 0.6 initially. I deleted the enitre Firebird folder and
my default profile. I installed Firebird 0.6.1 and the same problem occured.
