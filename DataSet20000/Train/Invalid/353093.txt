Grammatical error in Mozilla Firefox (use of "can't")
User-Agent:       Mozilla/5.0 (X11; U; Linux i686; en-US; rv:1.8.0.5) Gecko/20060731 Ubuntu/dapper-security Firefox/1.5.0.5
Build Identifier: Mozilla/5.0 (X11; U; Linux i686; en-US; rv:1.8.0.5) Gecko/20060731 Ubuntu/dapper-security Firefox/1.5.0.5

When browsing an Internet or Intranet web site via HTTP or FTP, Mozilla Firefox version 1.5.x displays the following error message on the user's screen: "Firefox can't find the server at www.sitethatdoesnotexist.com."

The error "Firefox can't find the server at www.sitethatdoesnotexist.com." is grammatically incorrect.

Reproducible: Always

Steps to Reproduce:
1. Visit a Internet location (via HTTP or FTP) that does not exist
2. Wait for Mozilla Firefox to timeout
3. Notice the error message
Actual Results:  
When browsing an Internet or Intranet web site via HTTP or FTP, Mozilla Firefox version 1.5.x displays the following error message on the user's screen: "Firefox can't find the server at www.sitethatdoesnotexist.com."

Expected Results:  
The error given by Firefox for Internet or Intranet locations that cannot be visited by HTTP or FTP should be given as "Firefox cannot find the server at www.sitethatdoesnotexist.com." Notice the difference from the error "Firefox can't find the server at www.sitethatdoesnotexist.com." as can't has been replaced by cannot for the error message given.
