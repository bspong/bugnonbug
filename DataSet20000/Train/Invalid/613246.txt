Moonlight is not compatible with Firefox 4
User-Agent:       Mozilla/5.0 (X11; U; Linux i686; it; rv:1.9.2.12) Gecko/20101026 Firefox/3.6.12
Build Identifier: Mozilla/5.0 (X11; Linux i686; rv:2.0b8pre) Gecko/20101118 Firefox/4.0b8pre

It is impossible to install any version of Moonlight on Firefox 4.
The compatibility of the "standard" version of the plugin

http://www.go-mono.com/moonlight/download.aspx

stops to Firefox 3.6.*

For the "preview" version

http://go-mono.com/moonlight/prerelease.aspx

the compatibility stops to 3.7

Disabling the extension compatibility check makes the it installable, but the associated plugin does not work.

Reproducible: Always
