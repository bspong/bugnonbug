(accidentally) Clicking both touch pad buttons on a tab closes it without warning in FF 1.5
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8) Gecko/20051107 Firefox/1.5
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8) Gecko/20051107 Firefox/1.5

On a Toshiba laptop Windows XP system with an Alps Electric pointing device (as well as driver) the simultaneous clicking of both touch pad buttons is set so that it enables autoscrolling in applications.

However when (accidentally) clicking both buttons over a tab it closes the tab without warning in FF 1.5

Reproducible: Always

Steps to Reproduce:
1. Set touch pad buttons to auto scroll when both are simultaneously clicked
2. Click both buttons over a tab
Actual Results:  
closes the tab

Expected Results:  
does nothing
