RoboForm Toolbar loads slowly when opening new tab & tabs flicker
User-Agent:       Mozilla/5.0 (Windows NT 5.2; rv:2.0) Gecko/20100101 Firefox/4.0
Build Identifier: Mozilla/5.0 (Windows NT 5.2; rv:2.0) Gecko/20100101 Firefox/4.0

FF 3.x works fine with RoboForm, issue cannot reproduced.
When running RoboForm 7.2.6 (and earlier releases) in FF 4.0 then the tabs starts flickering randomly, especially when opening new tabs.
Also, the loading time of the RF toolbar is very slow when opening new tabs.




Reproducible: Always

Steps to Reproduce:
1. Install FF 4.0
2. Install latest RoboForm
3. Open new tabs
4. watch toolbar behaviour
Actual Results:  
toolbar acts slowly, toolbar flickers

Expected Results:  
browser should not be slowed down

Reported issue to vendor of RoboForm. They have seen a spike in such cases but are unsure if it has something to do with the graphics driver or not.
