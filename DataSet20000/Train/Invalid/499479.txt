Dragging a link or unlinked URL onto the Tab Bar causes the link to be opened in both the current tab and new tab
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 6.0; en-US; rv:1.9.1) Gecko/20090616 Firefox/3.5
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 6.0; en-US; rv:1.9.1) Gecko/20090616 Firefox/3.5

Dragging a link or unlinked URL onto the Tab Bar causes the link to be opened in both the current tab and new tab when the proper behavior should be the link or URL loading only in a new tab.

Reproducible: Always

Steps to Reproduce:
1. Select the following url: http://www.mozilla.org/
2. Drag and drop the URL onto an empty space of the tab bar.
Actual Results:  
The URL is loaded into a new tab and in the current (active) tab.

Expected Results:  
The URL is loaded into a new tab only.
