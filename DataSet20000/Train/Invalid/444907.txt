search box doesn't show results in new tab after pressing Ctrl+Enter
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9) Gecko/2008052906 Firefox/3.0
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9) Gecko/2008052906 Firefox/3.0

The search box in the toolbar displays the search results in the current active tab, and if you click the magnifying glass button while holding Ctrl, it will open in a new tab. However, if you press Ctrl+Enter, it still displays in the current tab.

Reproducible: Always

Steps to Reproduce:
1. Open Firefox 3, type some text into the search box.
2. Press Ctrl+Enter.
Actual Results:  
The results still appear in the current tab, just as if you pressed Enter without Ctrl.

Expected Results:  
The results should appear in a new tab, just as if you clicked the magnifying glass button while holding Ctrl.
