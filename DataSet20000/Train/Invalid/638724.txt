left clicking on a search box suggestion does not work (Google Toolbar Search)
User-Agent:       Mozilla/5.0 (Windows NT 6.1; rv:2.0b12) Gecko/20100101 Firefox/4.0b12
Build Identifier: Mozilla/5.0 (Windows NT 6.1; rv:2.0b12) Gecko/20100101 Firefox/4.0b12

When You try to click on a suggestion in the search box, nothing happens. Only the text input in the search box remains and no search happens. However, using the down/up arrow keys to navigate to the suggested search and hitting enter fills the the search box with the suggested term and the search results appear. 

Reproducible: Always

Steps to Reproduce:
1.Bring up FF4
2.Type search term in search box on top right. For example as I type "te" The suggested search terms are 'team bhp, teamviewer, ted, telugu news,' etc. Left clicking on any of these search suggestions, leaves "te" in the search box. Nothing happens 
3.However using the down arrow to navigate to the relevant search term and hitting enter performs the resultant search
Actual Results:  
At step two - "te" remains in the search box. browser stays in the same page

Expected Results:  
selected item should be searched for using the preferred search engine and results displayed.
