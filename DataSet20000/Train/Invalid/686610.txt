No definitive way to remove custom Toolbar
User Agent: Mozilla/5.0 (compatible; MSIE 9.0; Windows NT 6.1; Trident/5.0; SLCC2; .NET CLR 2.0.50727; .NET CLR 3.5.30729; .NET CLR 3.0.30729; Media Center PC 6.0; Tablet PC 2.0; .NET CLR 3.0.30618; InfoPath.2; SLCC1)

Steps to reproduce:

1. Open Firefox. Navigate to the View, select Toolbars and go to Customize Toolbar.
2. Hit Add New Toolbar, enter a name for the New Toolbar (hit Ok) and add some items to the New Toolbar. 
Repeat the step few times. After that hit Done.
3. Navigate to the View, select Toolbars and go to Customize Toolbar again. 
4. Hit Restore Default Set.


Actual results:

All New Toolbars will be removed together. No definitive way to remove a single custom Toolbar.


Expected results:

It should be possible to remove custom Toolbar of your choice (not just all together).
