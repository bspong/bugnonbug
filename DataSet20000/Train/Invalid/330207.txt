Attempt to make a folder in "Save Link As" crashes 1.5.0.2 nightly
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 5.2; en-US; rv:1.8.0.2) Gecko/20060310 Firefox/1.5.0.2
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.2; en-US; rv:1.8.0.2) Gecko/20060310 Firefox/1.5.0.2

Any attempt to make a new folder via right-click action in the "Save Link As" window crashes Firefox.


Reproducible: Always

Steps to Reproduce:
1. Right-click on any file link.
2. Left-click "Save Link As."
3. Right-click in the new window.
4. Attempt to scroll mouse over "New" -> "Folder" or left-click "Folder" in "New."

Actual Results:  
Firefox crashes.

Expected Results:  
Made a new folder and offered to rename it.

Reporting OS is the Windows XP x64 release build w/latest patches. Error does not appear to occur in other applications (Notepad & Paint.NET 2.6 were compared; neither exhibit this error). Reporting OS has the latest iFox theme.
