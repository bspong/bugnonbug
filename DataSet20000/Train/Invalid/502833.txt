Element Properties box doesn't close by typing Command+W

Build Identifier: Mozilla/5.0 (Macintosh; U; Intel Mac OS X 10.5; en-US; rv:1.9.1) Gecko/20090624 Firefox/3.5

Firefox 3.5: mouse over any image, do a right-click (secondary-click), and choose Properties. The Element Properties popup does not respond to Command+W to close it

Reproducible: Always

Steps to Reproduce:
1. Open Firefox 3.5
2. Mouse over any image
3. Right-click (secondary-click) to bring up contextual menu
4. Choose Properties
5. Element Properties popup does not respond to Command+W to close
Actual Results:  
Element Properties window stays open.

Expected Results:  
Element Properties window would be closed.
