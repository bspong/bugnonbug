When double-clicking on Gmail Notifier, Firefox opens and freezes
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 5.1; en-GB; rv:1.8.1) Gecko/20061010 Firefox/2.0
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-GB; rv:1.8.1) Gecko/20061010 Firefox/2.0

Whenever I double-click on the Gmail Notifier icon in my system tray, Firefox opens and freezes.

Reproducible: Always

Steps to Reproduce:
1. Install Gmail Notifier (from http://mail.google.com/mail/help/notifier/notifier_windows.html)
2. Make sure Firefox is the default browser, and set up Gmail Notifier to check your Gmail account.
3. Double-click on the Gmail Notifier icon in the system tray.
Actual Results:  
Firefox opens, and then freezes up.

Expected Results:  
Firefox should have navigated to Gmail and not frozen up.

This bug was not present in Firefox 1.0.7.
