Clear Recent History breaks current downloads
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 6.0; en-US; rv:1.9.1) Gecko/20090624 Firefox/3.5 (.NET CLR 3.5.30729)
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 6.0; en-US; rv:1.9.1) Gecko/20090624 Firefox/3.5 (.NET CLR 3.5.30729)

In FF 3.5 but not 3.0.11, selecting Tools - Clear Recent History and then clearing the history, including file downloads, terminates any current downloads.

Reproducible: Always

Steps to Reproduce:
1. Start a download
2. Clear Recent History - All history - include downloads

Actual Results:  
Any current downloads are terminated

Expected Results:  
Current download would not be terminated, would not be cleared from the list of downloads

Did not happen in FF 3.0.11.
