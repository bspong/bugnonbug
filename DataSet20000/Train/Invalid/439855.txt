The new feature added "Recently Added Bookmarks" changes its position at the bookmarks list everytime Firefox is inititialised. Even if it is set back to normal manually, it moves again when closing and opening.

Reproducible: Always

Steps to Reproduce:
1.Open Firefox and set the "recently added bookmarks" to its original postion.
2.Close Firefox.
3.Open Firefox again and check "recently added bookmarks" position.
Actual Results:  
New feature "recently added bookmarks" moved one step down the bookmarks list.

Expected Results:  
New feature "recently added bookmarks" should not move from its set position.

I am using Default Theme with some extensions like FlashGot, Gmail Notifier and Spanish dictionary.
