Vista: New Default Programs API's and settings better integrate Firefox into the Windows Shell
Vista supports a new set of API’s and registry settings for registering default applications, “related applications”, mime types, and settings related to the detailed product descriptions available to the user through the Default Programs applet on the Vista start menu. Using these methods to register Firefox will better integrate Firefox into the shell.
