Addon-bar close button not visible if toolbars set to 'text' mode instead of 'icons'
User-Agent:       Mozilla/5.0 (Windows NT 5.1; rv:2.0) Gecko/20100101 Firefox/4.0
Build Identifier: Mozilla/5.0 (Windows NT 5.1; rv:2.0) Gecko/20100101 Firefox/4.0

If in 'Customize toolbar' dialog, 'Text' is chosen instead of 'Icons' or 'Icons and text', the close button on addon-bar is not visible.

It is, however, still clickable and still closes the addon-bar.

Reproducible: Always

Steps to Reproduce:
1. Click View -> Toolbars -> Customize
2. Select 'Text' in 'Show:' options box
Actual Results:  
Close button on addon-bar disappears and is not replaced by text.

Expected Results:  
Instead of close icon, text saying 'Close' should appear.
