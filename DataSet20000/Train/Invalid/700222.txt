Firefox 7.0.1 not rendering gridlines in an online battleship game
User Agent: Mozilla/5.0 (X11; Linux i686; rv:7.0.1) Gecko/20100101 Firefox/7.0.1
Build ID: 20110928134238

Steps to reproduce:

Reproduce as follows:

1. Load Firefox 3.6.23 on 32-bit Linux
2. Navigate to http://www.auntiemomo.com/cakeordeath/battleship.html and start a game.  Click on a few areas on the left pane and see the grid lines.
3. Load Firefox 7.0.1
4. Navigate to the same site as above, and notice that there are no gridlines.




Actual results:

Firefox 7.0.1 is not drawing things that Firefox 3.6.23 draws.


Expected results:

Firefox 7.0.1 should be rendering gridlines on the http://www.auntiemomo.com/cakeordeath/battleship.html page, but is not.
