OK button doesn't work on Bookmark Properties dialog
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 5.0; en-US; rv:1.4b) Gecko/20030516 Mozilla Firebird/0.6
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.0; en-US; rv:1.4b) Gecko/20030516 Mozilla Firebird/0.6

Once bookmark Properties dialog is opened, it can't be closed with OK button (or
Enter, etc.); only cancel works and changes can't be saved.

Reproducible: Always

Steps to Reproduce:
1. Open Bookmark Manager
2. Right-click bookmark, choose Properties from context menu
3. Click "OK"

Actual Results:  
Nothing; dialog not dismissed, changes not saved

Expected Results:  
Save changed bookmark properties and dismiss dialog
