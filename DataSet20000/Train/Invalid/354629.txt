firefox menu and customize toolbar are remained disabled after customize toolbar is opened then closed

Reproducible: Always

Steps to Reproduce:
1. right click on any part of the firefox menu to open customize toolbar dialog box (you will notice that the firefox menu will be disabled.
2. close the customize toolbar dialog box 
3. the firefox menu is now inaccessible, including address bar dropdown and the right click shortcut for the customize toolbar itself
4. if you open a new instance of firefox (launch other firefox program) and repeat the step 1 and 2, and go back to the first firefox instance, the firefox bookmark toolbar will not load.

Actual Results:  
firefox menu disable

Expected Results:  
firefox menu must be enabled.
