An URL in the location bar is not updated after clicking a link and moving to the new location
User-Agent:       Mozilla/5.0 (Windows NT 5.1; rv:2.0.1) Gecko/20100101 Firefox/4.0.1
Build Identifier: Mozilla/5.0 (Windows NT 5.1; rv:2.0.1) Gecko/20100101 Firefox/4.0.1

If one clicks itTV link on the http://www.ittf.com/ site, firefox loads the new page http://www.ittf.com/itTV, but doesn't update location bar. It shows just http://www.ittf.com/ and reload button reloads just the http://www.ittf.com/ page.

Reproducible: Always

Steps to Reproduce:
1. Enter and go in the location bar 
2. Click itTV link
3. Firefox should load http://www.ittf.com/ittv/ page
4. But the location bar still displays http://www.ittf.com/  

Actual Results:  
The location bar is not updated after moving from http://www.ittf.com/ to http://www.ittf.com/ittv/

Expected Results:  
The location bar should be updated with the http://www.ittf.com/ittv/ URL
