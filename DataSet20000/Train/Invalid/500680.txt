Navigation toolbar suddenly expands and becomes unusable after logging into Yahoo! mail or logging out
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9.1) Gecko/20090624 Firefox/3.5
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.9.1) Gecko/20090624 Firefox/3.5

The navigation toolbar suddenly expands as shown in this picture: http://img196.imageshack.us/img196/9843/navigationtoolbar.jpg, after logging into Yahoo! mail. If this happens, logging out will cause all of Firefox to blank out and no longer respond. If this doesn't happen when logging into Yahoo! mail, then logging out will cause this to happen.

Reproducible: Always

Steps to Reproduce:
1. Go to www.yahoo.com
2. Press 'mail'
3. Log into Yahoo! mail
4. (If this bug doesn't show up) Press logout
Actual Results:  
The navigation toolbar suddenly expands and becomes unusable.

Expected Results:  
The navigation toolbar shouldn't have had any problems after logging out.
