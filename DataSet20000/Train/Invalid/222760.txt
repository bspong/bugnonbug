The Help button in the Filter Rules window does not open any help file.

Reproducible: Always

Steps to Reproduce:
Go to Tools / Message Filters... In the Message Filters dialog box, click the
New... button. This opens the Filter Rules dialog. Click the Help button or
press F1.
Actual Results:  
Nothing

Expected Results:  
Opens up help.
