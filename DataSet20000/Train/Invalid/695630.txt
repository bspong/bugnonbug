Window contents do not resize on window resize in xmonad
User Agent: Mozilla/5.0 (X11; Linux x86_64; rv:8.0) Gecko/20100101 Firefox/8.0
Build ID: 20111011233032

Steps to reproduce:

1) Open firefox
2) Open any site
3) Resize window to smaller one


Actual results:

Window resized but all contents and interface that were on right end of the window disappeared (vertical scrollbar, addonbar contents, new tab button etc.). Page contents did not resize. No horizontal scrollbar appeared.
Bottom elements moced upper as they should


Expected results:

Right aligned interface items should remain visible. Horizontal scrollbar should appear. Interface and page should resize
