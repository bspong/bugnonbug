FlashGot 0.7.2. unable to detect the Download Manager
** Credit goes to Marcia for the find **

The most recent version of FlashGot, 0.7.2, displays "??? NO DOWNLOAD MANAGER DETECTED ???" on Windows Vista using trunk Minefield.

Build ID: Mozilla/5.0 (Windows; U; Windows NT 6.0; en-US; rv:1.9b3pre) Gecko/2008011105 Minefield/3.0b3pre

Steps to Reproduce:

1. With a trunk build on Windows Vista, install FlashGot 0.7.2 from https://addons.mozilla.org/en-US/firefox/addon/220
2. Right-click on an element/link, and select "FlashGot Options ->" and look at its flyout menu

Actual Results:

The flyout menu displays the following:

"??? NO DOWNLOAD MANAGER DETECTED ???"

This message also shows up from clicking the "Options" button in the FlashGot 0.7.2 add-on from within the Add-ons Manager, under FlashGot's "General" tab.
