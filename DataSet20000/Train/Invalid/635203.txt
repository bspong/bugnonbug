Firefox resizes when opened from Taskbar preview
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 6.1; en-US; rv:1.9.2.13) Gecko/20101203 Firefox/3.6.13
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 6.1; en-US; rv:1.9.2.13) Gecko/20101203 Firefox/3.6.13

I have set browser.taskbar.previews.enable;true.
When I click on any tab preview FF resizes.
It will not maintain maximize position. 

Also sometimes taskbar preview will not show updated tabs.


Reproducible: Always

Steps to Reproduce:
1.Set browser.taskbar.previews.enable;true 
2.Maximize FF
3.Open multiple tabs
3.Minimize the FF
4.Click on the tab preview in taskbar
5.Check FF window size.
Actual Results:  
Resize the FF window size

Expected Results:  
Restore previous FF window size
