Status bar disappears in full-screen mode (Mac OS X only)
User-Agent:       Mozilla/5.0 (Macintosh; U; PPC Mac OS X Mach-O; en-US; rv:1.7.5) Gecko/20041107 Firefox/1.0
Build Identifier: Mozilla/5.0 (Macintosh; U; PPC Mac OS X Mach-O; en-US; rv:1.7.5) Gecko/20041107 Firefox/1.0

When the browser window is put into full screen mode, the Status Bar disappears
from the bottom of the window.  It reappears when the window is put back into
normal mode.

Reproducible: Always
Steps to Reproduce:
1. Start the browser
2. Make sure that the Status Bar is visible (View->Status Bar)
3. Put the window in Full Screen mode (View->Full Screen)

Actual Results:  
The Status Bar disappears

Expected Results:  
The Status Bar should stay in place, as it does on Windows and Linux
