Tabs show as Windows
User-Agent:       Mozilla/5.0 (Windows NT 6.1; WOW64; rv:2.0) Gecko/20100101 Firefox/4.0
Build Identifier: Mozilla/5.0 (Windows NT 6.1; WOW64; rv:2.0) Gecko/20100101 Firefox/4.0

One Firefox window with 3 tabs open shows as 3 windows on the Windows taskbar. This did not occur in version 3 or  version 4 RC's.



Reproducible: Always

Steps to Reproduce:
1.Open Firefox
2. open a new tab
3. open another new tab
Actual Results:  
Windows shows stacked Firefox icons in the taskbar which normally indicates multiple windows, not tabs in a window.

Expected Results:  
Only one icon should show per instance of FF
