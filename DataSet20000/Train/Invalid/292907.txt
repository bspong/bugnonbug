Trackpoint scrolling does not work with IBM 3.3.0 Drivers.
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.7.7) Gecko/20050414 Firefox/1.0.3
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.7.7) Gecko/20050414 Firefox/1.0.3

When trying to scroll on a page that scrolling is allowed, when using IBM 3.3.0
Trackpoint drivers, FireFox does not allow scrolling. When updating to IBM 3.50,
this corrects the problem, and FireFox allows scrolling. 

Reproducible: Always

Steps to Reproduce:
1. Go to webpage that scrolling is allowed
2. Press the 3rd mouse button located on keyboard called the Trackpoint Scroller
3. Does not work.
4. Go to IBM.com
5. Update Trackpoint Drivers
6. Restart computer
7. Be happy you can scroll.

Actual Results:  
This works now, due to updated Trackpoint Drivers. FireFox show allow more older
versions of this driver to be allowed to scroll to acompany a wide range of users.

Expected Results:  
Let the older driver version scroll using FireFox.
