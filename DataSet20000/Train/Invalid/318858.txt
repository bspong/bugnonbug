Bookmarks menu is empty on startup
User-Agent:       Mozilla/5.0 (X11; U; Linux i686; en-US; rv:1.8) Gecko/20051202 Firefox/1.5
Build Identifier: Mozilla/5.0 (X11; U; Linux i686; en-US; rv:1.8) Gecko/20051202 Firefox/1.5

When Firefox is first started, the Bookmarks menu displays only the top three options: Bookmark This Page, Bookmark All Tabs, Manage Bookmarks.  No bookmarks or folders appear.  If Manage Bookmarks is selected and then the window immediately closed, the bookmarks reappear as usual.

Reproducible: Always

Steps to Reproduce:
1. Close and reopen Firefox
2. Select the Bookmarks menu.
3. Select Manage Bookmarks.
4. Close the Manage Bookmarks window.
5. Select the Bookmarks Menu again

Actual Results:  
After Step 2 the menu is unpopulated.  After step 5 is is repopulated.

Expected Results:  
The bookmarks should appear after step 2.
