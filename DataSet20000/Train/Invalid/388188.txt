hasFeatures not showing support for MathML
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8.1.4) Gecko/20070515 Firefox/2.0.0.4
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8.1.4) Gecko/20070515 Firefox/2.0.0.4

MathML has significant support (at least in FFx 2.0.0.4), but the hasFeature interface reports false. 

Reproducible: Always

Steps to Reproduce:
1. Test for MathML support using document.implementation.hasFeature("org.w3c.dom.mathml", null);

Actual Results:  
hasFeature returns false.

Expected Results:  
hasFeature should return true.

Related to https://bugzilla.mozilla.org/show_bug.cgi?id=30534 . Note that as of this writing, MathML support in FFx 3 is not certain (see https://bugzilla.mozilla.org/show_bug.cgi?id=324857 ), but if it is supported to a significant degree, hasFeature should return true.
