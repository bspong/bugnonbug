Flash doesn't work and javascript error with this doctype: http://www.w3.org/TR/html4/loose.dtd
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 5.1; pt-BR; rv:1.8) Gecko/20051025 Firefox/1.5
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; pt-BR; rv:1.8) Gecko/20051025 Firefox/1.5

When a website uses the doctype: <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">

The flash plugin does not work properly and if there is an ad block using javascript to publish the flash ad, the browser will show part of the javascript code and not the ad.

when the doctype is not specified, you'll no longer se the error: http://baixaki.ig.com.br/imagens/shots1/baixaki.ig.htm

Reproducible: Always

Steps to Reproduce:
1. The page with doctype specified and the error concerning flash, just access the website http://baixaki.ig.com.br, if you dont see the error, refresh the page until a flash ad comes.
2. the page without the error, we've removed the doctype: http://baixaki.ig.com.br/imagens/shots1/baixaki.ig.htm
