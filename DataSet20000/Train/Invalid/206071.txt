Dialog box buttons do not fit in box when using Stardock WindowBlinds
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 5.0; en-US; rv:1.4b) Gecko/20030516 Mozilla Firebird/0.6
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.0; en-US; rv:1.4b) Gecko/20030516 Mozilla Firebird/0.6

WindowBlinds is a theming utility for Windows, http://www.windowblinds.net. When
using WindowBlinds, the dialog box buttons do not fit inside the dialog box.
This happens regardless of Mozilla theme being used, and does not occur when
WindowBlinds is not used.

Reproducible: Always

Steps to Reproduce:
1. Install/enable WindowBlinds
2. Open a Mozilla dialog box.
3. Enjoy guessing which button is supposed to do what.

Actual Results:  
Can see just the top edge of the buttons.

Expected Results:  
Correctly sized the dialog box, or permitted the user to resize it themselves.
