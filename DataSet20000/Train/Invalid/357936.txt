'open all in tabs' on folder loses existing tabs
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8.1) Gecko/20061010 Firefox/2.0
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8.1) Gecko/20061010 Firefox/2.0

When opening a folder of bookmarks from the toolbar menu (right click - open all in tabs or middle click folder), it uses existing tabs and closes any others.

Reproducible: Always

Steps to Reproduce:
1. Right click on a folder in the bookmarks toolbar menu & select "Open all in tabs" or Middle click on a folder in the bookmarks toolbar menu

Actual Results:  
Existing tabs are used and any others are closed.

Expected Results:  
Should use new tabs instead.
