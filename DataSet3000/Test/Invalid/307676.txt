Removed Home button reappears on restart
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8b4) Gecko/20050908 Firefox/1.4
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.8b4) Gecko/20050908 Firefox/1.4

After customizing the Navigation toolbar and removing the Home button, the Home
button reappears after exiting and restarting Firefox.

Reproducible: Always

Steps to Reproduce:
1. Customize Navigation toolbar
2. Remove Home button
3. Exit Firefox
4. Restart Firefox

Actual Results:  
Home button reappears

Expected Results:  
Home button should stay removed
