Input box in a new backgrounded tab is selected
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 5.0; rv:1.7.3) Gecko/20041001 Firefox/0.10.1
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.0; rv:1.7.3) Gecko/20041001 Firefox/0.10.1

After Middle Clicking a link to Gmail or Google, the input box on Gmail or
Google is selected from the backgrounded tab. From the existing tab you can type
in a new search, press enter, and the backgrounded tab will carry out the search.

Reproducible: Always
Steps to Reproduce:
1. Open a page with a standard link to gmail or google
2. Middle Click the link (New tab)
3. Enter a search query and press enter

Actual Results:  
The backgrounded tab carried out teh search

Expected Results:  
Not accepted input as there was no input boxes selected from the foreground tab.
