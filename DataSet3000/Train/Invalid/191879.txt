"Get New Themes" is clickable but doesn't appear as a link
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.3a) Gecko/20030201 Phoenix/0.5
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; en-US; rv:1.3a) Gecko/20030201 Phoenix/0.5

The "Get New Themes" link in the Themes section of the new Options/Preferences
dialog has the appearance of normal text, but should actually look like a hyperlink.

Reproducible: Always

Steps to Reproduce:
1. Go to Tools > Options > Themes
2. Hover mouse over "Get New Themes"
3. Left-click on "Get New Themes"

Actual Results:  
"Get New Themes" has the appearance of normal text, and when hovering, the mouse
appears as a text selection cursor.  However, the link to the Themes page still
works.

Expected Results:  
"Get New Themes" should be highlighted and underlined like a normal hyperlink,
and when hover, the mouse cursor should appear as a pointing hand.
