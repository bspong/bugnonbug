Firefox window automatically switch from Maximize to restore
User-Agent:       Mozilla/5.0 (Windows; U; Windows NT 5.1; rv:1.7.3) Gecko/20041001 Firefox/0.10.1
Build Identifier: Mozilla/5.0 (Windows; U; Windows NT 5.1; rv:1.7.3) Gecko/20041001 Firefox/0.10.1

When open many tabs with diferent webpages in the same Firefox window the window
switch itelf from maximize to restore (non maximized window.)


Reproducible: Always
Steps to Reproduce:
1. Open one maximized Firefox window
2. Open many tabs in that window
3. When you open some webpages the Firefox window switch itself from Maximized
to restore (non maximized window)
4. This hapens only in some webpages I can't give you some examples of such
webpages that cause this bug because when I have open many tabs with diferent
webpages could'nt know exactly wich webpage cause this bug, but it did hapen
many times that Firefox switch itself from Maximized to restore (non maximized
window)

Actual Results:  
Maximized Firefox window switch itself to restore (non maximized window) when
open many tabs with diferent webpages on them.

Expected Results:  
Firefox window should maintain its maximized status.

It is a bug that should be fixed.
