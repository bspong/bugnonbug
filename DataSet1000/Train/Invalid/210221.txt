Installation of All-In-One gestures disables contextual menus
User-Agent:       Mozilla/5.0 (Macintosh; U; PPC Mac OS X Mach-O; en-US; rv:1.5a) Gecko/20030620 Mozilla Firebird/0.6
Build Identifier: Mozilla/5.0 (Macintosh; U; PPC Mac OS X Mach-O; en-US; rv:1.5a) Gecko/20030620 Mozilla Firebird/0.6

Installing the All-In-One Gestures package disables contextual menus, regardless
of which mouse button is set as the gesture activator.

Reproducible: Always

Steps to Reproduce:
1. Install All-In-One gestures
2. Attempt to summon the contextual menu in the browser window using either
control+click or the right button of a multi-button mouse.


Actual Results:  
Contextual menus do not appear for pages or images; they do work on Tabs.

Expected Results:  
Appearance of the contextual menu.
