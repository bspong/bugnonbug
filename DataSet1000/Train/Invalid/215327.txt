correctly calibrating monitor resolution causes dialog box fonts to become unreasonably small

Dialog box fonts are acceptable if I configure a 72 dpi or 96 dpi resolution.
Configuring my monitor's correct resolution (102 dpi) causes them to be
unreasonably small.

Reproducible: Always

Steps to Reproduce:
Tools/Options/Display resolution, set size greater than 100 dpi

Actual Results:  
dialog boxes became nearly unreadable

Expected Results:  
continued to display dialog box fonts at their proper size
